# MEV/MRV 个体差异系统

> 来源：The Powerlifting Program Design Manual, "Determining Volume" 章节
> 
> 交叉引用：计划设计阶段 MRV 审计时，`references/volume-recovery/mrv-audit.md` 会引用本文档的个体差异调整方法。

---

## 为什么需要个体差异？

MEV（最小有效容量）和 MRV（最大可恢复容量）**不是固定值**。两个同样体重、同样训练水平的训练者，MEV/MRV 可能相差 30% 以上。

JTS 官方手册列出了 9 大个体差异因素，每个因素都独立影响训练者的容量需求。

---

## 9 因素汇总表

| 因素 | 对 MEV 的影响 | 对 MRV 的影响 | 机制/说明 |
|------|-------------|-------------|----------|
| **性别（女性）** | ↑ 提高 | ↑ 提高 | 更小的肌肉质量 + 较低睾酮 → 每次训练疲劳更少 → 需更高容量 |
| **体重大** | ↓ 降低 | ↓ 降低 | 更大肌肉质量 → 更多肌肉损伤 → 恢复更慢；但高体脂可能反向提高 MEV |
| **身高高** | ↓ 降低 | ↓ 降低 | 更长的杠铃行程 → 每组做功更多 → 单组更疲劳 |
| **力量大** | ↑ 提高 | ↓ 降低 | 越强壮 → 每组刺激越大 → 需要更少就能维持，但 MRV 反而更低（"强者更难维持"悖论） |
| **经验多** | ↑ 提高 | ↑ 提高 | 更大的专项工作能力；但超过 12 年持续高强度训练后，MRV 可能小幅下降 |
| **年龄大** | ↑ 提高 | ↓ 降低 | 恢复能力下降 → 需要更多刺激才能维持进步，但总容量上限更低 |
| **饮食差** | ↑ 提高 | ↓ 降低 | 分解代谢环境 → 需要更多训练来克服；但总恢复能力受限 |
| **睡眠差（<5h）** | ↑ 提高 | ↓ 降低 | 恢复效率降低 → 需要更多刺激；但总容量上限显著降低 |
| **生活压力大** | ↑ 提高 | ↓ 降低 | 身体无法区分训练压力和生活压力 → 总压力负载上升 |

---

## 各因素详解

### 1. 性别

女性训练者的肌肉质量通常更小，睾酮水平更低，导致：
- 每次训练对肌肉的绝对破坏更少
- 需要更高的总容量才能达到相同的刺激效果
- **结论：女性需要比男性更多的组数**

### 2. 体重

体重大 → 肌肉量多（通常）→ 更多肌肉需要恢复：
- 更大肌肉质量 = 更多肌肉损伤 = 更长的恢复时间
- 但同时更大的肌肉质量也更能"保持"（持有适应）
- **例外**：高体脂训练者可能 MEV 升高（分解代谢环境）

### 3. 身高

杠铃行程越长 → 每组做功越多 → 单组疲劳更大：
- 180cm vs 165cm 的训练者，同样做 100kg 深蹲 5 次，180cm 训练者做的物理功多约 10%
- **结论：高个子需要比矮个子更少的组数**

### 4. 力量水平

这是一个"悖论"因素：
- 越强壮 → 每组训练的质量（刺激）越高 → MEV 提高
- 但越强壮 → 每组对 CNS 和肌肉的破坏也越大 → MRV 降低
- **这是高级训练者训练计划设计的核心难点**

### 5. 经验水平

- 初级：对刺激反应敏感，少量训练就能进步，但容易过度训练
- 中级：需要更多容量来继续进步，工作能力提升
- 高级（>12年持续高强度训练）：每个组都是极高刺激，MRV 可能下降
- **关键转折点**：经验增长到一定程度后，MRV 趋势反转

### 6. 年龄

- 生物年龄本身提供的信息有限（同样 45 岁的人差异巨大）
- 其他 8 个因素通常已经涵盖了年龄带来的差异
- 在所有因素相等的情况下：更年轻 → 更高 MRV，更低 MEV

### 7-9. 饮食 / 睡眠 / 压力

这三个**生活方式因素**有相似的规律：
- **差**的状态 → 需要更多训练来克服分解代谢（MEV ↑）→ 但恢复能力更差（MRV ↓）
- **好**的状态 → 需要更少训练就能进步（MEV ↓）→ 且能承受更多（MRV ↑）
- 睡眠 <5h 和 >7h 之间的差异尤甚显著

## 如何使用此系统

### 设计计划时

收集用户的个体差异信息后，对 MRV 表进行方向性调整：

```
示例1：180cm、95kg 的男性中级训练者
→ 基准胸部 MRV：22 组/周
→ 体重 95kg（偏高，-1） + 身高 180cm（中等，0） + 男性（基准）
→ 调整后 MRV：约 20-21 组/周

示例2：160cm、55kg 的女性新手
→ 基准胸部 MRV：22 组/周
→ 女性（+3-4） + 体重 55kg（偏低，+1-2） + 新手（略低 MEV，但不影响 MRV 上限）
→ 调整后 MRV：约 26-28 组/周
```

### 审计已有计划时

当用户反馈以下问题时，用此系统重新评估：
- "计划容量有点大/不够" → 检查个体差异因素是否被正确考虑
- "我已经训练了 X 年，还需要这么多组吗？" → 检查经验因素
- "最近睡眠/压力不好，要不要调整？" → 临时降低 MRV

---

## 重要提醒

> ⚠️ **方向性而非精确性**：本表格提供的是调整方向和大致幅度。精确的 MEV/MRV 需要通过**训练日志数据逐步校准**——如果连续 2-3 周都在预定容量下按计划进步，说明当前容量在有效区间内。
>
> ⚠️ **动态性**：MEV/MRV 不是固定的——会随着训练进展、体重变化、生活方式变化而改变。至少每 2-3 个训练周期重新评估一次。
