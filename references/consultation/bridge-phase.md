# Bridge Phase（过渡期）

> 来源：The Powerlifting Program Design Manual, "Bridge Phases"
>
> 交叉引用：当用户询问周期之间如何安排，或两个正式训练周期之间需要过渡时，参考本文档。

---

## 什么是 Bridge Phase

Bridge Phase 是两个正式训练周期（mesocycle）之间的 **1-2 周过渡期**。它与减载周不同——减载是在单个周期内的恢复，Bridge Phase 是周期之间的"刷新"。

### Bridge Phase vs 减载周

| 特征 | 减载周（Deload） | Bridge Phase |
|------|---------------|-------------|
| **位置** | 在一个 8 周周期内（通常是 W4） | 在两个周期之间 |
| **目的** | 让身体从容量期的高容量中恢复 | 心理休息、身体康复、探索新动作 |
| **内容** | 降低容量/强度，但仍做比赛动作 | 可以做完全不同类型的训练 |
| **持续时间** | 1 周 | 1-2 周 |
| **频率** | 每 3-4 周高压训练后 | 每 8-12 周周期结束后 |

---

## Bridge Phase 的四大目的

1. **心理休息**：长期专注于三大项和严格的周期化训练可能导致精神倦怠。Bridge Phase 让你"玩一玩"。

2. **小伤病康复**：长期训练累积的关节不适、肌腱炎症等可以在 Bridge Phase 中愈合——因为你在用不同的动作模式。

3. **避免适应性抵抗**：长期做同样的动作会导致身体对该刺激的适应效率下降（负面反馈循环）。尝试新动作可以打破这种适应。

4. **准备下一个周期**：身体和精神都焕然一新后，下一个周期的训练质量会更高。

---

## Bridge Phase 的设计原则

JTS 官方手册给出了四大设计原则：

| 原则 | 说明 | 示例 |
|------|------|------|
| **让运动员开心** | 做新的、有趣的、很久没做的活动；设定短暂的临时目标 | 尝试壶铃训练、挑战自己的引体向上数量 |
| **做运动员** | 保持运动性——跑、跳、投、负重行走 | 冲刺（推荐 hills 或 sleds）、投掷（药球）、跳跃训练、农夫行走 |
| **创造进步** | 超负荷原则依然适用，但不必是重量——可以是更多 reps、更快速度、更短休息 | 上周做了 5 组农夫行走 50m，这周 6 组 |
| **不要忘记细节** | 依然保持 10-20 分钟的呼吸练习、纠正性训练等基础工作 | 死虫式、腹式呼吸、肩袖肌群激活 |

## 适用的训练内容

Bridge Phase 的训练可以非常多样化：

- **Strongman 风格**：农夫行走、沙袋搬运、轮胎翻转、圆木推举
- **Athletic Performance**：冲刺（短距离）、药球投掷、跳箱
- **代谢挑战**：轻量的 CrossFit 风格分解训练（高代谢需求但不破坏肌肉）
- **Bodybuilding 风格**：平时不做的孤立训练（高位下拉、腿弯举等）
- **灵活性/活动度**：Goata 训练、动物流、瑜伽

---

## 适用场景

| 场景 | Bridge Phase 时长 | 强度 |
|------|-----------------|------|
| 两个 8 周周期之间 | 1 周 | 中等——保持运动但不积累疲劳 |
| 赛季/比赛结束后 | 1-2 周 | 低到中等——优先恢复 |
| 长期使用同一套动作，感觉"腻了" | 1 周 | 中等——重点是新鲜感 |
| 有小伤病需要恢复（肌腱炎等） | 1-2 周 | 低——给身体时间愈合 |
| 精神倦怠，训练欲望低 | 1 周 | 做有趣的事——重点是享受运动 |

---

## 常见误区

1. ❌ "Bridge Phase = 完全不练" → 完全停练 1-2 周会导致力量和体能显著下降
2. ❌ "Bridge Phase 就是多做有氧" → 有氧会额外激活 AMPk（见 `fatigue-sources.md`），与下一个周期的力量训练目标冲突
3. ❌ "Bridge Phase 不需要结构" → 仍然需要超负荷原则——找个方式创造进步
4. ❌ "Bridge Phase 和减载是一回事" → 减载是在周期内，Bridge 是周期之间；减载做比赛动作的轻版，Bridge 做完全不同的事情

---

## 计划中的位置

一个包含 Bridge Phase 的典型年度安排：

```
周期1（8周）→ Bridge Phase（1周）→ 周期2（8周）→ Bridge Phase（1-2周）→ 周期3（8周）→ 比赛/测试
```

> 如果你的 8 周周期以测试周结束，Bridge Phase 尤为重要——因为测试周后的身心都需要刷新。
