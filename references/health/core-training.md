## 核心稳定训练（Core Stability Training）

### 为什么核心稳定很重要？

**价值**：
1. **力量传递**：核心刚性 → 力量从腿/臀传递到杠铃
2. **伤病预防**：核心弱 → 下背疼痛/伤病（详见 `references/health/injury-prevention.md`）
3. **动作质量**：深蹲/硬拉/卧推都需要核心稳定

---

### 核心训练分类

| 类型 | 动作示例 | 训练目标 | 安排建议 |
|------|---------|---------|---------|
| **抗伸展（Anti-Extension）** | 平板支撑（Plank）、腹轮（Ab Wheel）、山羊挺身（Back Extension） | 防止下背过度伸展 | 每次训练后：3×30-60秒 |
| **抗旋转（Anti-Rotation）** | Pallof Press、鸟狗式（Bird-Dog）、死虫式（Dead Bug） | 防止躯干旋转 | 每次训练后：3×10-15/侧 |
| **抗侧屈（Anti-Lateral Flexion）** | 侧平板（Side Plank）、农夫行走（Farmer's Walk） | 防止躯干侧屈 | 每次训练后：3×30秒/侧 |
| **爆发力（Power）** | 药球投掷（Medicine Ball Throw）、绳索劈砍（Cable Chop） | 核心爆发力 | 容量期：2×10 |
| **负重（Loaded Carries）** | 农夫行走、负重行走（Waiter's Walk） | 全身稳定性 | 容量期/力量期：3×20-40米 |

---

### 核心训练在周期中的安排

#### 容量期 W1-3

**目标**：打基础，低强度高容量

**推荐组合**：
```
每次训练后：
1. 平板支撑：3×45秒
2. 鸟狗式：3×12/侧
3. 侧平板：3×30秒/侧
```

#### 力量期 W5-6

**目标**：维持核心稳定，不堆容量

**推荐组合**：
```
每次训练后：
1. 腹轮：3×10
2. Pallof Press：3×12/侧
3. 农夫行走：3×20米（中等重量）
```

#### 冲刺期 W7-8

**目标**：降容保恢复，核心训练维持但不疲劳

**推荐组合**：
```
每次训练后：
1. 平板支撑：2×30秒（轻量）
2. 死虫式：2×10/侧
```

---

### 核心训练的 MRV 审计

**核心肌群 MRV 基准**：≈ 15-20 组/周（所有核心训练合计）

**审计示例**：
```
用户核心训练审计：
- 平板支撑：3组
- 鸟狗式：3组
- 腹轮：3组
- 农夫行走：3组
- 总计：12组 < 15-20组（MRV）✅ 通过
```

**调整原则**：
- 核心训练**不易过度**：过度核心训练 → 下背疲劳 → 影响主项表现
- **优先保证主项恢复**：冲刺期核心训练降容

---

## 常见错误

### 核心训练
1. ❌ 核心训练做到力竭 → ✅ 核心训练 RPE 6-7，避免过度疲劳
2. ❌ 冲刺期核心训练不降容 → ✅ 冲刺期核心训练降容 50%
3. ❌ 忽略抗旋转/抗侧屈 → ✅ 核心训练要全面（抗伸展/抗旋转/抗侧屈）
