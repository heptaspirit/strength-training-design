## 热身（Warm-up）流程

### 通用热身流程（推荐）

```
1. 全身升温（5-10 min）
   - 跑步机 / 划船机 / 跳绳
   - 目标：轻微出汗，心率上升至 100-120 bpm

2. 动态拉伸（5-10 min）
   - 关节活动度练习（肩、髋、踝、脊柱）
   - 不做静态拉伸（静态拉伸应在训练后做）

3. 热身组（针对主项）
   - 用 50% → 60% → 70% 1RM 做 2-3 组 × 5 次
   - 目的：激活神经系统，找到当日感觉
```

### 按身高/肢体比例调整热身

| 体型 | 热身重点 |
|------|----------|
| **高个子** | 更多踝关节、髋关节动态拉伸；肩关节稳定性练习 |
| **矮个子** | 更多胸椎、肩关节活动度练习 |
| **长腿短躯干** | 重点激活踝关节和髋关节 |
| **短腿长躯干** | 重点激活胸椎和肩关节 |


## 关节灵活度注意事项

### 深蹲相关

**踝关节灵活度**：
- 测试：靠墙蹲，脚尖离墙 10cm，能否蹲下且膝盖触墙
- 不足时：脚跟垫杠铃片（2.5-5kg），或做踝关节动态拉伸
- 高个子更易出现踝关节灵活度不足

**髋关节灵活度**：
- 测试：坐姿，双腿前伸，能否保持腰椎中立
- 不足时：增加髋关节动态拉伸，考虑宽站距深蹲

**胸椎灵活度**：
- 测试：坐姿，双手抱头，能否向两侧旋转躯干
- 不足时：影响深蹲上半身前倾角度，增加下背部压力


### 卧推相关

**肩关节灵活度**：
- 测试：靠墙站立，双手上举，能否贴墙且腰椎不超伸
- 不足时：影响卧推握距和肩胛骨稳定性
- 卧推前做肩关节圆周运动、弹力带拉肩

**胸椎灵活度**：
- 胸椎僵硬 → 卧推时肩膀前倾，增加肩部受伤风险
- 建议：卧推前做胸椎滚动（泡沫轴）


### 硬拉相关

**髋关节灵活度**：
- 测试：单腿硬拉，能否保持平衡且髋关节不歪斜
- 不足时：硬拉 startup 易圆背，增加受伤风险

**踝关节灵活度**：
- 传统硬拉对踝关节要求较低
- 相扑硬拉对踝关节灵活度要求较高（需更大髋关节外展）


## 放松（Cool-down）流程

### 训练后放松（推荐）

```
1. 静态拉伸（10-15 min）
   - 每个主要肌群拉伸 30-60 秒
   - 重点：股四头肌、腘绳肌、胸肌、背阔肌

2. 筋膜放松（可选）
   - 泡沫轴：股四头肌、IT Band、上背部
   - 痛点保持 30-60 秒

3. 呼吸放松
   - 仰卧，双手放腹部，缓慢深呼吸 5-10 次
   - 帮助神经系统从"战斗模式"切换到"恢复模式"
```

### 按训练日类型调整

| 训练日类型 | 放松重点 |
|-----------|----------|
| **深蹲日** | 股四头肌、髋关节、踝关节静态拉伸 |
| **卧推日** | 胸肌、肩关节、胸椎泡沫轴放松 |
| **硬拉日** | 腘绳肌、下背部、髋关节放松 |
| **高容量日** | 延长放松时间至 20 min，多加筋膜放松 |


## 日常灵活度训练建议

### 每周 2-3 次灵活度训练（非训练日或训练后）

**推荐动作**：

1. **踝关节灵活度**
   - 靠墙踝背屈练习：3×10 次/侧
   - 台阶踝下落：3×10 次/侧

2. **髋关节灵活度**
   - 90/90 髋关节拉伸：3×30 秒/侧
   - 青蛙趴（Frog Stretch）：3×30 秒

3. **胸椎灵活度**
   - 泡沫轴胸椎滚动：3×10 次
   - 猫牛式（Cat-Cow）：3×10 次

4. **肩关节灵活度**
   - 门框拉伸（胸肌）：3×30 秒/侧
   - 弹力带肩外旋：3×15 次/侧


## 身高与灵活度

| 身高 | 常见灵活度短板 | 训练建议 |
|------|---------------|----------|
| **高个子** | 踝关节、髋关节 | 每次训练前加强踝关节动态拉伸 |
| **矮个子** | 胸椎、肩关节 | 加强胸椎灵活度训练 |
| **肢体比例不均** | 根据长短腿/长短臂调整 | 针对性加强相关关节灵活度 |

## 常见错误

1. ❌ 训练前做大量静态拉伸 → ✅ 训练前做动态拉伸，静态拉伸留给训练后
2. ❌ 忽略踝关节灵活度对深蹲的影响 → ✅ 高个子尤其需要关注踝关节灵活度
3. ❌ 硬拉后不放松下背部 → ✅ 硬拉日必须放松腘绳肌和下背部
4. ❌ 灵活度训练只做一次 → ✅ 灵活度需要持续、规律的练习才能改善
