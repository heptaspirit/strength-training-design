# Westside → JTS 方法论整合脉络

**来源**: The Westside Barbell Book of Methods (Louie Simmons, 2007)  
**定位**: Westside 是 JTS 方法论的重要源头之一。JTS 保留其核心框架，加入 MEV/MRV、RPE 自我调节和个体差异系统，使其更适合自然训练者。

---

## 1. 整体关系：Westside 是"原型"，JTS 是"进化版"

| 维度 | Westside (2007) | JTS 继承了什么 | JTS 改良了什么 |
|------|----------------|---------------|---------------|
| **周期化类型** | 并发周期化（Concurrent） | ✅ 保留：同周发展多种素质 | ➕ 加入容量期→减载→力量期→冲刺期的**阶段划分** |
| **强度控制** | 基于经验的百分比（60-90%） | ✅ 保留：百分比区间作为参考 | ➕ 加入 **RPE 自我调节**，尊重日内波动 |
| **容量管理** | 高容量（14次训练/周） | ❌ 不采用：自然训练者无法恢复 | ➕ 用 **MEV/MRV 系统**量化个体容量边界 |
| **动作轮换** | 每周轮换主项变式 | ✅ 保留：变式多样化 | ➕ 轮换周期延长（4-6周），减少适应 |
| **疲劳管理** | 通过动作轮换 + GPP | ✅ 保留：GPP 主动恢复 | ➕ 加入 **减载周、疲劳上限审计** |
| **个体差异** | 未明确讨论 | ❌ 不采用 | ➕ 9因素个体差异系统（性别/体重/经验等） |

**结论**: JTS = Westside 框架 × MEV/MRV 安全边界 × RPE 自我调节。Westside 是"怎么练"，JTS 补充了"练多少"和"什么时候停"。

---

## 2. Zatsiorsky 三大方法 → JTS 的应用

Westside 基于 Zatsiorsky (1995) 的三大方法：

| 方法 | Westside 应用 | JTS 保留/改良 |
|------|-------------|-------------|
| **Maximal Effort** (最大努力法) | 每周 2 次 Max Effort Day (上肢+下肢) | ✅ 保留为 **TS (Top Set)**，但限于力量期/冲刺期（W5-8），容量期不做 |
| **Dynamic Method** (动态法) | 60-80% + 最大速度 | ✅ 保留为 **速度训练**，但不再强制每周固定日，可按需融入 |
| **Repetition Method** (重复法) | 辅助动作 8-12 次至力竭 | ✅ 保留用于辅助动作，但加入**双进阶**（先加次数再加重），避免力竭 |

## 3. 动作分类系统：Westside 核心，JTS 全部保留

这是 JTS 从 Westside 继承的最完整框架：

```
Westside 三层分类：

  Main Exercise (主项)
  ├── 比赛动作或直接变式
  └── 例: Box Squat, Board Press, Rack Pull

  Supplemental Exercise (补充)
  ├── 针对特定薄弱环节
  └── 例: Close-grip Bench, Anderson Squat

  Accessory Exercise (辅助)
  ├── 孤立肌群 + 预防损伤
  └── 例: Triceps Pushdown, Face Pull, Hamstring Curl
```

**JTS 如何应用**：详见 `references/westside/exercise-classification.md`

**关键差异**：
- Westside: Main Exercise **每周更换**
- JTS: Main Exercise **每 4-6 周更换**（减少适应，便于追踪进度）

## 4. 波浪形加载 → JTS 的阶段化周期

### Westside 的波浪（3-5-1）

```
W1: 3 次 @ 75-80%  →  容量 + 技术
W2: 5 次 @ 70-75%  →  肌肥大
W3: 1 次 @ 95%+    →  最大力量
W4: 回到 W1 + 2.5-5kg
```

### JTS 的阶段化保留

| JTS 阶段 | 对应的 Westside 波浪元素 |
|----------|----------------------|
| 容量积累期 W1-3 (5×8 @ RPE 6-7) | ≈ 波浪的"5次"阶段（肌肥大） |
| 减载期 W4 (降容 40-50%) | JTS **新增**，Westside 没有明确的减载周 |
| 力量建立期 W5-6 (TS + BO) | ≈ 波浪的"3次"阶段 + 最大努力法 |
| 冲刺期 W7-8 (TS 继续递增) | ≈ 波浪的"1次"阶段 |

**JTS 的改良**：将 Westside 的 3 周微周期扩展为 8 周中周期，用**减载周**取代 Westside 依赖**动作轮换**作为疲劳管理手段。

## 5. 特殊力量素质 → JTS 的薄弱环节分析

Westside 定义 4 种特殊力量：

| 力量素质 | 短板表现 | JTS 如何解决 |
|----------|---------|-------------|
| **启动力量** | 卧推底部推不起 | 选择 Starting Strength 类型变式（停顿卧推、Deficit Deadlift） |
| **加速力量** | 卧推中间段变慢 | 加入弹力带/铁链（可变阻力匹配力量曲线） |
| **绝对力量** | 上不了大重量 | 增加 TS 频率（W5-8，但上限 RPE 8.5） |
| **反应力量** | 深蹲底部"弹"不起来 | 加入弹震式训练或 Plyometrics |

**JTS 如何应用**：详见 `references/westside/special-strengths.md`

**关键差异**：
- Westside: 通过**每周轮换不同动作**来覆盖所有特殊力量
- JTS: 通过**分析用户具体短板**来选择针对性动作，效率更高

## 6. 可变阻力（弹力带/铁链）

**核心原理**（Westside 和 JTS 一致）：弹力带/铁链在底部阻力最小、顶部最大，匹配人体的自然力量曲线。

**JTS 的态度**：可选工具，但不是必须。JTS 更强调**动作本身的选择**（如选择停顿卧推而不是弹力带卧推），因为动作变式对力量曲线的影响大于可变阻力。

**实用建议**：如果用户有弹力带/铁链可用，在 W5-6 力量期加入 2-3 周可变阻力训练，有助于突破 sticky point。如果没有，通过选择合适的变式动作替代。

---

## 7. GPP 与恢复

**Westside 的核心逻辑**（JTS 完全保留）：
- GPP 不是"辅助训练"，而是**恢复手段**
- Sled Drag、农夫行走、医用球投掷：低强度、高容量、促进血液循环
- CNS 恢复：高强度训练后的主动恢复

**JTS 补充**：GPP 安排在**休息日**或**训练日之后**，不占用主项训练时间。

---

## 8. 为什么 JTS 更适合自然训练者

| Westside 的局限 | JTS 的解决 |
|----------------|-----------|
| 每周 2 次 Max Effort（过高频率） | TS 只出现在 W5-8，且上限 RPE 8.5 |
| 高容量辅助训练（恢复不及） | MRV 审计确保不超个体恢复边界 |
| 依赖经验（缺乏个体差异考虑） | 9 因素个体差异系统 + RPE 自我调节 |
| 无明确的减载机制 | 每 4 周一次减载周（W4, W8…） |
| 主项每周更换（追踪困难） | 主项每 4-6 周更换，便于追踪进度 |
