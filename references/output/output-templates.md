## 退阶方案

【自我调节原则（JTS）】
- TS 当天 RPE 比预期高 2 点以上 → BO 组数减 1 组，保 TS 质量
- 连续 2 周 RPE 超标 → 插入额外减载周
- 生活压力/睡眠差 → 主动降容 20-30%，无需等到 RPE 超标

【BO 组逐级减重指引】
- 即便 RPE 很低，BO 组练到第四、五组时也可能会特别累（累积疲劳）
- 建议：BO 组第 3 组起，可视情况主动减重 2.5-5kg（约 TS 重量的 3-5%）
- 或让训练者根据感觉自行调整：第 3 组感觉吃力 → 第 4 组主动降重 5%
- 示例（TS 100kg）：
  - BO 第 1-2 组：92.5kg（92.5% TS）
  - BO 第 3 组：90kg（90% TS，主动减重）
  - BO 第 4-5 组：87.5kg（87.5% TS，进一步减重）
- 原则：**BO 组是"积累容量"，不是"测试极限"**，感觉累就减，不要硬撑

【重量步进】
- 最小哑铃片：XXkg → 重量取 XXkg 的整数倍
- 示例：参考重量 93.2kg，步进 2.5kg → 取 92.5kg 或 95kg

## 使用说明

1. **选择模板**：根据用户训练天数选择对应模板
   - 3 天/周 → 模板二
   - 4 天/周 → 模板一（推荐）
   - 5 天/周 → 模板三（高级）
   - 2 天/周 → 建议用户增至 3 天（ACSM 建议每肌群至少 2x/周）

2. **填写重量**：根据 RPE 参考表和动态 PR 调整计算 TS/BO 重量

3. **MRV 审计**：输出计划后必须附上 MRV 审计表

4. **训练日安排**：务必询问用户偏好，不要假设"周一深蹲、周三卧推"
   - 询问："你希望哪天练哪个主项？有没有需要避开的时段（如健身房人流高峰）？"

5. **重量步进**：根据用户健身房哑铃片配置确定重量取整方式
   - 默认最小 1.25kg 片 → 步进 2.5kg
   - 有 0.5kg 片 → 步进 1kg
   - 生成计划前必须确认用户健身房配置

6. **辅助动作递增方式**：辅助动作（夹胸、面拉、腿伸展等）不使用"每周+2.5kg"，而使用**双进阶（Double Progression）**：
   - 设定次数范围（如 8-12 次）
   - 先在当前重量增加次数（3×8 → 3×10 → 3×12）
   - 到达上限后加重 2.5-5kg，回到下限
   - 模板中辅助动作处标注「双进阶」

7. **Cluster Set 备选**：力量期/冲刺期 TS 目标 RPE ≥ 8.5 时，提供 Cluster Set 备选方案：
   - 如「TS 1×5 @ RPE 8.5」→ 备选「Cluster 5×1 @ 同重量，组间 20s」
   - 模板中 TS 行标注「或 Cluster 5×1」

### 使用建议

1. **随手记**：训练完立刻填，不要等回家再回忆（数据会不准）
2. **RPE 校准**：连续 3 周 RPE 实际与计划偏差 > 1 → 可能是你的 RPE 感知需要校准
3. **异常标记**：只要出现疼痛、抽筋、严重疲劳，立即记录
4. **趋势 > 单次**：单次 RPE 超标不用慌，看连续 2-3 周的趋势
