>

> 2. 从外部权威信息中提取（如 JTS、Reactive Training Systems 官方资料）
> 3. 明确告知用户：此信息来自文档外，仅供参考
> 
> **原则**：本文档是"核心知识库"，但不是"全部知识库"。

> **⚠️ 重要提醒（务必阅读）**
> 标准 RPE 表是**通用参考值**，基于人群平均数据（Reactive Training Systems / Zourdos et al. 2016）。
> **每个人的 RPE 感知存在差异**——有人习惯性高估，有人习惯性低估。
> 训练时应以**自身感受为主**，RPE 表仅作起始参考，后续必须根据实际 RPE 反馈进行调节。
> 不要盲目照搬表格数值。

## 三、RPE 使用指南

### 场景 1：规定重量计算（设计训练计划时用）

如果计划规定某组 "5 次 @ RPE 8"，查表得 RPE 8 / 5 次 = 80% 1RM。
用参考 PR × 80% = 目标重量。

```
示例：
深蹲参考 PR = 140kg，计划 W5 TS 5次 @ RPE 8
→ TS 重量 = 140kg × 80% = 112kg
```

### 场景 2：回顾性 1RM 估算（不用实测 1RM 也能追踪进步）

完成一组后记录 RPE，即可估算当前 1RM：

```
示例：
深蹲 120kg × 3次，RPE 9
查表：RPE 9 / 3次 = 90% 1RM
→ 估算 1RM = 120kg ÷ 0.9 ≈ 133kg
```

> ⚠️ 估算误差约 5-10%，仅供参考，不能替代实测 1RM。

### 场景 3：根据当日状态调节重量（RPE 的核心价值）

RPE 的最大优势是**兼容每日状态波动**（睡眠、压力、恢复等）。
如果平时 80% 1RM 是 RPE 8，但今天做完感觉是 RPE 9 → 说明状态差，后续组减重或减容量。

## 四、不同训练目标的 RPE 范围

| RPE 范围 | 训练目标 | 说明 |
|-----------|---------|------|
| 9-10 | 峰值化（Peaking） | 比赛前准备；单/双次接近极限。每周 1-2 次。神经疲劳大，容量低。 |
| 8-9 | 力量（Strength） | 主项力量训练；3-6 次。每周 2-3 次。在足够恢复的前提下建立最大力量。 |
| 6-8 | 肌肥大（Hypertrophy） | 肌肉增长；6-12 次。每周 3-5 次。高容量，恢复能力范围内尽量堆量。 |
| 5-7 | 技术练习 / 减载 | 技术打磨和主动恢复。关注杆速和动作质量，几乎不积累疲劳。 |

## 五、⚠️ 重要：个体差异与注意事项

### 1. RPE 感知的个体差异

标准 RPE 表是**人群平均值**。实际使用中：

- 有人习惯性**高估** RPE（实际还能做 3 次，但感觉是 RPE 8）
- 有人习惯性**低估** RPE（实际只能做 1 次，但感觉是 RPE 7）
- **新手往往 RPE 判断不准**，需要 2-3 个月训练才能建立准确的 RPE 感知

**建议做法**：
- 前 4 周计划可先用固定百分比（如 RPE 8 ≈ 80% 1RM）作为起点
- 每 2 周做一次 AMRAP 测试，校准个人 RPE 感知
- 训练 3 个月后，RPE 数据会趋于准确

### 2. 渐进超负荷幅度因人而异（不要生搬模板）

模板里写的"每周 +2.5-5kg"是**参考值，不是铁律**：

- 新手（<1 年）：可能 8 周深蹲进步 20-30kg
- 中级（1-3 年）：通常 8 周深蹲进步 10-15kg
- 高级（3-5 年）：通常 8 周深蹲进步 5-10kg
- 精英（>5 年）：可能 8 周深蹲仅进步 2.5-5kg

**设计计划前务必询问用户**：
- "上一轮周期三大项各进步了多少？"
- "这轮周期希望/预期进步多少？"

根据用户的实际进步速度调整参考 PR 的递增幅度。

### 3. 重量步进：确认健身房哑铃片配置

不同健身房的哑铃片配置差异很大，直接影响计划重量的可行性：

| 哑铃片配置 | 最小步进 | 适用场景 |
|-----------|---------|---------|
| 最小 1.25kg 片 | 2.5kg | 标准商业健身房 |
| 有 0.5kg 片 | 1kg | 竞技/精密训练 |
| 用户自备小片 | 0.5kg 或更小 | 经验丰富的力量举运动员 |

**设计计划前务必询问用户**：
- "你健身房的哑铃片最小是多少？"
- "你有没有自备小哑铃片（0.5kg 或更小）？"

如果用户的健身房只有 1.25kg 片（最小步进 2.5kg），计划重量应取 2.5kg 的整数倍。
如果用户有 0.5kg 片，可以实现 1kg 步进，计划可以更精密。

### 4. 1RM 估算的局限性

用 AMRAP + RPE 估算 1RM 的误差约 5-10%，仅供追踪趋势，不能替代实测 1RM。
特别是在设计冲刺期/测试周重量时，若有条件，建议做一次实测 1RM。

## 六、渐进超负荷与动态 PR 调整

### 问题背景

设计 8 周（或更长）周期计划时，用户的 1RM 不是静态的。例如：
- **初始状态**：卧推 1RM = 100kg
- **目标状态**：卧推 1RM = 105kg（8 周后）

如果整个计划都基于初始 PR 100kg 计算重量，W5-8 的重量会偏保守，无法充分刺激进步。

### 解决方案：动态 PR 调整

在周期计划中，**参考 PR 应随周期进度逐步提升**，以反映用户的预期进步。

#### 调整方法（线性渐进，需根据用户情况调整幅度）

| 周期阶段 | 参考 PR | 说明 |
|---------|---------|------|
| **容量期 W1-3** | 初始 PR | 用户当前真实 1RM |
| **减载周 W4** | 初始 PR + 2.5-5kg | 减载期保持技术，重量略增 |
| **力量期 W5-6** | 初始 PR + 2.5-5kg | 用户力量已有所提升 |
| **冲刺期 W7-8** | 目标 PR - 2.5-5kg | 接近目标，但保留余量 |
| **测试周 W9** | 目标 PR | 测试新 1RM |

> ⚠️ 上表递增幅度为**通用参考**。实际递增幅度需根据用户上轮周期的实际进步情况调整。
> 不要生搬上表数值。

#### 调整方法（分阶段渐进，适用于 12/16 周周期）

| 周期阶段 | 参考 PR | 说明 |
|---------|---------|------|
| **容量期 W1-4** | 初始 PR | 前 4 周用初始 PR |
| **减载周 W5** | 初始 PR + 2.5-5kg | 第一次减载 |
| **容量期 W6-9** | 初始 PR + 2.5-5kg | 中期进步 |
| **减载周 W10** | 初始 PR + 5-7.5kg | 第二次减载 |
| **力量/冲刺期 W11-12** | 目标 PR - 2.5-5kg | 接近目标 |

## 八、RPE 自我调节与重量调整

### 训练当天的 RPE 反馈循环

即使计划设计了参考 PR 和预期 RPE，**训练当天的实际 RPE 可能偏离预期**。此时需要自我调节。

**调节规则（JTS 原则）**：

| 实际情况 | 预期 | 调整方案 |
|---------|------|---------|
| 实际 RPE < 预期 RPE | 计划 RPE 8，实际 RPE 7.5 | 下周 TS 增重 2.5-5kg |
| 实际 RPE = 预期 RPE | 计划 RPE 8，实际 RPE 8 | 维持或小幅增重 1.25-2.5kg |
| 实际 RPE > 预期 RPE + 1 | 计划 RPE 8，实际 RPE 9+ | 下周 TS 减重 2.5-5kg，或减少 BO 组数 |

### 示例：根据实际 RPE 调整后续训练

**场景**：用户 W5 深蹲 TS 计划 117.5kg @ RPE 8，但实际 RPE 9（太重了）

**分析**：可能参考 PR 设置过高，或用户恢复不足

**调整方案**：
1. **W6 深蹲 TS 减重**：117.5kg → 115kg（降低 2.5kg）
2. **检查恢复情况**：睡眠、压力、营养是否不足
3. **如连续 2 周 RPE 超标**：插入额外减载周

## 九、AMRAP + RPE 估算动态 PR

### 为什么需要动态估算？

如果用户在周期中做了 AMRAP 测试，可以用结果**更新参考 PR**，使后续计划更准确。

### 估算公式

```
方法一（Epley Formula）：
1RM ≈ 重量 × (1 + 次数 / 30)

方法二（RPE 表）：
1RM ≈ 重量 / (%1RM 对应 RPE 和次数)
```

### 动态更新参考 PR 的时机

| 时机 | 操作 |
|------|---------|
| **容量期结束后（W4 减载周前）** | 做一次 AMRAP 测试，更新参考 PR |
| **力量期中（W5 或 W6）** | 如感觉计划太轻，可做 AMRAP 测试 |
| **冲刺期前（W7 前）** | 做一次 AMRAP 测试，确认参考 PR 是否接近目标 |

### 示例：根据 AMRAP 测试结果更新参考 PR

**背景**：
- 用户初始 PR：卧推 100kg
- W4 减载周前做 AMRAP 测试：100kg × 6 次，RPE 9
- 估算 1RM：100 ÷ 0.95 = 105.3kg ≈ 105kg

**操作**：
- 更新参考 PR：100kg → 105kg
- W5-6 力量期基于 105kg 计算 TS 重量（而非初始 100kg）

## 十、总结：重量计算的标准流程

设计计划时，遵循以下流程：

1. **确定用户初始 PR 和目标 PR**
2. **询问用户上轮周期进步情况**，确定参考 PR 调整幅度（不要生搬模板的 +2.5-5kg）
3. **询问用户健身房哑铃片配置**，确定重量步进（2.5kg 还是 1kg）
4. **制定参考 PR 调整计划**（见"六、渐进超负荷与动态 PR 调整"）
5. **根据参考 PR 和计划 RPE，查表计算 TS 重量**（使用"一、完整 RPE 对照表"）
6. **计算 BO 重量**：BO 重量 = TS 重量 × (1 - 8~12%)
7. **在计划中注明参考 PR**（让用户知道重量是如何计算的）
8. **训练当天根据实际 RPE 自我调节**（JTS 原则）

**计划中表示方式示例**：

```
W5 卧推 TS：
- 参考 PR：102.5kg（初始 100kg + 预期进步 2.5kg）
- 计划 RPE：8（5 次 @ 80% 1RM）
- TS 重量：102.5kg × 80% = 82kg → 取 82.5kg（2.5kg 步进）
- BO 重量：82.5kg × 90% = 74.3kg → 取 75kg（3 组 × 5 次）
```

## 进阶渐进超负荷方法

### 双进阶（Double Progression）

**概念**：不直接加重量，先在当前重量下增加次数，达到次数上限后再加重量。适合辅助动作和肌肥大训练。

**为什么需要？** 简单"每周 +2.5kg"对主项有效，但辅助动作（如夹胸、面拉）重量空间有限，直接加重量可能导致：
- 技术变形（重量太大做不了标准动作）
- 安全风险（孤立动作大重量更危险）
- 忽略肌肥大窗口（次数范围也很重要）

**规则**：

```
设定一个次数范围（如 8-12 次）
↓
第一步：先用较轻重量做到范围下限（3×8）
↓
第二步：每次训练尝试多加次数（3×8 → 3×9 → 3×10 → 3×11 → 3×12）
↓
第三步：当所有组都能做到 3×12 时 → 加重量 2.5-5kg
↓
第四步：回到 3×8，重新开始
```

**示例**（夹胸 3×8-12 @ RPE 7）：
```
W1: 20kg 3×8  →  RPE 6.5（轻松，需增加次数）
W2: 20kg 3×10 →  RPE 7（刚好进入范围）
W3: 20kg 3×12 →  RPE 7.5（到达上限，下次加重量）
W4: 22.5kg 3×8 → RPE 7（回到范围下限）
```

**适用场景**：
- ✅ 辅助孤立动作（夹胸、面拉、腿伸展、侧平举）
- ✅ 核心训练
- ✅ 肌肥大阶段所有辅助
- ❌ 主项 TS（主项用 %1RM + RPE 周期递增）

---

### Cluster Set（组簇训练）

**概念**：把一个重组的次数拆成多个"迷你组"，每组之间只休息 15-30 秒。保持大重量，但降低单组的疲劳和 RPE。

**原理**：磷酸原系统（ATP-CP）在 15-30 秒内可恢复 50-70%，所以"拆开做"可以维持大重量但降低单组 RPE。

**规则**：

```
常规组：1×5 @ RPE 8.5, 120kg（5 次连做）
↓
Cluster：5×1 @ 同重量 120kg，组间休息 20 秒（5 个单次，各短休后做）
↓
效果：总次数相同，但每组 RPE 降至 6-7
```

**进阶变式**：

| Cluster 类型 | 格式 | 休息 | 适用 |
|-------------|------|------|------|
| **单次簇** | 5×1 @ 20s 休息 | 20s | 力量期/冲刺期 TS 替代 |
| **双次簇** | 3×2 @ 20-30s 休息 | 20-30s | 力量期 BO |
| **递减簇** | 1×3 + 1×2 + 1×1（同重量） | 30s | 冲刺期补充容量 |

**示例**（深蹲力量期 W5）：
```
原方案：TS 1×5 @ RPE 8.5, 117.5kg
Cluster 方案：TS 5×1 @ RPE 6-7, 117.5kg（组间 20s）

收益：
- 总训练量相同（都是 5 次）
- 每组 RPE 从 8.5 降至 6-7
- CNS 疲劳更可控
- 每次都是"第一次"的质量
```

**适用场景**：
- ✅ **力量期/冲刺期 TS**：想练大重量但担心 RPE 超标
- ✅ **突破平台期**：同一重量卡住了，拆成 Cluster 继续练
- ✅ **恢复不足的训练日**：身体状况差但不想降重量
- ✅ **技术练习**：每次只做 1-2 次，质量更高
- ❌ **容量期**：容量期不需要大重量，Cluster 没意义
- ❌ **新手**：需要先建立连续做组的能力

**Cluster Set 的 CNS 疲劳优势**：

| 对比维度 | 常规 1×5 @ RPE 8.5 | Cluster 5×1 @ RPE 6-7 |
|---------|-------------------|----------------------|
| 总训练量 | 相同 | 相同 |
| 单组 RPE | 8.5（高） | 6-7（低） |
| CNS 疲劳 | 高（连续大重量） | 低（分散短休） |
| 技术质量 | 后 2-3 次可能变形 | 每次都是最好技术 |
| 总耗时 | 短 | 略长（+1-2 min） |

## 常见错误

1. ❌ 整个计划用初始 PR 计算所有重量 → ✅ 参考 PR 应随周期进度逐步提升
2. ❌ 忽略训练当天的 RPE 反馈 → ✅ RPE 是自我调节的核心工具
3. ❌ AMRAP 测试后不更新参考 PR → ✅ 根据测试结果动态调整后续计划
4. ❌ 参考 PR 调整幅度过大（如一周 +10kg）→ ✅ 保守渐进（通常每周 +1.25-2.5kg，根据具体用户调整）
5. ❌ 盲目照搬 RPE 表数值 → ✅ 以自身感受为主，RPE 表仅作参考
6. ❌ 不考虑健身房哑铃片配置 → ✅ 设计计划前询问用户哑铃片最小步进
