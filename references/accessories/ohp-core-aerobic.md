## 实力推（Overhead Press, OHP）

### 为什么在力量训练中加入 OHP？

**价值**：
1. **肩部整体发展**：前束/中束/后束 + 肩袖肌群
2. **核心稳定性**：推举时核心必须刚性支撑
3. **全身协调性**：与深蹲/硬拉形成「全身力量三角」
4. **功能性**：日常生活中推物过头顶的能力

**适用人群**：
- 希望全面发展上半身力量
- 非纯力量举竞技（力量举只有卧推，无 OHP）
- 希望改善肩部稳定性和健康

---

### OHP 变式与辅助动作

| 动作名称 | 中文 | 技术要点 | 适用场景 |
|---------|------|---------|---------|
| **Overhead Press (OHP)** | 实力推（站姿推举） | 站姿，核心刚性，臀/腿驱动辅助 | 主项或辅助 |
| **Push Press** | 借力推 | 腿驱动 + 髋伸展辅助，能推更大重量 | 力量期/冲刺期辅助 |
| **Push Jerk** | 炸弹推（借力推举） | 推至头顶后，髋/膝快速弯曲吸能（接铃） | 奥举风格，技术难度高 |
| **Superman Press** | 超人推 | 俯身姿势，强调肩袖稳定性 | 康复/预防性训练 |
| **Seated Dumbbell Press** | 坐姿哑铃推举 | 消除腿驱动，纯肩部力量 | 容量期辅助 |
| **Pavlo Press** | 帕夫罗推 | 空杆或轻重量，肩袖预热 | 每次训练前热身 |
| **Face Pull** | 面拉 | 绳索拉向面部，强化肩袖和后束 | 每次训练后辅助 |

---

### OHP 在周期中的安排

#### 作为主项（替代卧推或加入训练日）

**适用用户**：非力量举竞技，希望全面发展上半身

**安排示例**（4天/周）：
```
周一：深蹲 + OHP + 上身拉
周三：卧推 + 暂停卧推 + 夹胸
周五：硬拉 + OHP（轻） + 划船
周日：上身推辅助 + 核心 + 有氧
```

#### 作为辅助动作（大多数力量举运动员）

**容量期 W1-3**：
- OHP：3×8 @ RPE 7（作为肩部辅助）
- 或 Push Press：3×5 @ RPE 7-8（爆发力）

**力量期 W5-6**：
- OHP：2×5 @ RPE 8
- 或 Push Press：2×3 @ RPE 8.5

**冲刺期 W7-8**：
- OHP：1×3-5 @ RPE 6-7（降容保恢复）
- 或移除 OHP，优先主项恢复

---

### OHP 技术要点

#### 标准 OHP（站姿实力推）

**动作阶段**：
1. **起始**：杠铃置于锁骨/前肩，握距略宽于肩，核心收紧，臀部夹紧
2. **推起**：腿/臀微驱动（不是借力推），杠铃贴脸直线推起
3. **锁定**：肘部完全伸展，头部微前伸（让杠铃通过），核心保持刚性
4. **下放**：控制下放，杠铃贴脸回到锁骨

**关键提示**：
- **核心刚性**：推举时臀部不能松动，腰背不能过度伸展（避免腰椎压力）
- **腿驱动 vs. 借力推**：OHP 只允许「微驱动」，Push Press 才是明显腿驱动
- **杠铃轨迹**：贴脸直线向上，不是弧线

#### Push Press（借力推）

**与 OHP 的区别**：
- **明显腿驱动**：髋/膝快速伸展，将杠铃「弹」离肩膀
- **重量更大**：通常比 OHP 多 10-20% 重量
- **技术性**：需要协调腿/髋/肩/臂的爆发力

**适用场景**：
- 力量期/冲刺期辅助（爆发力）
- **不能替代 OHP**：Push Press 更多是「全身爆发力」，OHP 是「纯肩部力量」

#### Push Jerk（炸弹推/借力推举）

**与 Push Press 的区别**：
- Push Press：推至锁定位置（手臂完全伸展）
- Push Jerk：推至头顶后，**髋/膝快速弯曲**（接铃），类似高翻的接铃动作

**技术难度**：高（需要奥举技术基础）

**建议**：除非有奥举经验，否则优先选 Push Press

---

### OHP 的 MRV 审计

**肩部 MRV 基准**：≈ 24 组/周

**审计示例**：
```
用户：男性，30岁，训练3年
OHP 安排：3×8 @ RPE 7（容量期）

肩部容量审计：
- OHP：3组
- 帕夫罗推：3组（热身）
- 面拉：3组（辅助）
- 总计：9组 < 24组（MRV）✅ 通过
```

**调整原则**：
- 若肩部容量接近 MRV（≥80%）→ 减少面拉或帕夫罗推组数
- OHP 作为主项时，占用更多 MRV 配额

## 核心稳定训练（Core Stability Training）

### 为什么核心稳定很重要？

**价值**：
1. **力量传递**：核心刚性 → 力量从腿/臀传递到杠铃
2. **伤病预防**：核心弱 → 下背疼痛/伤病
3. **动作质量**：深蹲/硬拉/卧推都需要核心稳定

---

### 核心训练分类

| 类型 | 动作示例 | 训练目标 | 安排建议 |
|------|---------|---------|---------|
| **抗伸展（Anti-Extension）** | 平板支撑（Plank）、腹轮（Ab Wheel）、山羊挺身（Back Extension） | 防止下背过度伸展 | 每次训练后：3×30-60秒 |
| **抗旋转（Anti-Rotation）** | Pallof Press、鸟狗式（Bird-Dog）、死虫式（Dead Bug） | 防止躯干旋转 | 每次训练后：3×10-15/侧 |
| **抗侧屈（Anti-Lateral Flexion）** | 侧平板（Side Plank）、农夫行走（Farmer's Walk） | 防止躯干侧屈 | 每次训练后：3×30秒/侧 |
| **爆发力（Power）** | 药球投掷（Medicine Ball Throw）、绳索劈砍（Cable Chop） | 核心爆发力 | 容量期：2×10 |
| **负重（Loaded Carries）** | 农夫行走、负重行走（Waiter's Walk） | 全身稳定性 | 容量期/力量期：3×20-40米 |

---

### 核心训练在周期中的安排

#### 容量期 W1-3

**目标**：打基础，低强度高容量

**推荐组合**：
```
每次训练后：
1. 平板支撑：3×45秒
2. 鸟狗式：3×12/侧
3. 侧平板：3×30秒/侧
```

#### 力量期 W5-6

**目标**：维持核心稳定，不堆容量

**推荐组合**：
```
每次训练后：
1. 腹轮：3×10
2. Pallof Press：3×12/侧
3. 农夫行走：3×20米（中等重量）
```

#### 冲刺期 W7-8

**目标**：降容保恢复，核心训练维持但不疲劳

**推荐组合**：
```
每次训练后：
1. 平板支撑：2×30秒（轻量）
2. 死虫式：2×10/侧
```

---

### 核心训练的 MRV 审计

**核心肌群 MRV 基准**：≈ 15-20 组/周（所有核心训练合计）

**审计示例**：
```
用户核心训练审计：
- 平板支撑：3组
- 鸟狗式：3组
- 腹轮：3组
- 农夫行走：3组
- 总计：12组 < 15-20组（MRV）✅ 通过
```

**调整原则**：
- 核心训练**不易过度**：过度核心训练 → 下背疲劳 → 影响主项表现
- **优先保证主项恢复**：冲刺期核心训练降容

## 适度有氧训练（Moderate Aerobic Training）

### 为什么力量训练者需要有氧？

**价值**：
1. **恢复加速**：有氧促进血液流动，帮助代谢废物清除
2. **心血管健康**：纯力量训练者心血管风险可能增加
3. **体能基础**：改善训练中的组间恢复
4. **减脂**：如需减脂，有氧是必要补充

---

### 有氧训练类型与选择

| 类型 | 示例 | 强度 | 适用场景 |
|------|------|------|---------|
| **低强度稳态（LISS）** | 快走、慢跑、自行车、游泳 | RPE 3-5（可对话） | 恢复日/减载周 |
| **中等强度（MISS）** | 慢跑、椭圆机、划船机 | RPE 5-6（微喘） | 非训练日/训练后 |
| **高强度间歇（HIIT）** | 冲刺、波比跳、战绳 | RPE 8-9（力竭） | **不建议**力量训练者（干扰恢复） |
| **日常活动（NEAT）** | 走路、爬楼梯、站立 | RPE 2-3 | 每天 |

---

### 有氧训练在周期中的安排

#### 容量期 W1-3

**目标**：维持基础体能，不干扰恢复

**推荐安排**：
```
选项1（推荐）：非训练日 LISS 30-45分钟
- 例如：周二/周四/周六 快走或自行车

选项2：训练后 LISS 10-15分钟
- 例如：深蹲训练后，自行车 10 分钟（促进腿部恢复）

避免：HIIT（干扰容量期恢复）
```

#### 力量期 W5-6

**目标**：维持有氧，不干扰力量表现

**推荐安排**：
```
选项1（推荐）：非训练日 LISS 20-30分钟
- 例如：周二/周四 快走或椭圆机
- 特别注意：TS（Top Set）训练日后的第二天不要安排有氧

选项2：轻量训练日后 LISS 10-15分钟
- 例如：卧推日（上肢为主）训练后，椭圆机 10 分钟
- 注意：力量期 TS 重量大，恢复更重要，有氧需更保守

选项3：单独有氧日（如用户时间够多）
- 例如：周日只做有氧 20-30 分钟 LISS
- 适用：恢复能力强的用户（精英级/年轻男性）
```

#### 冲刺期 W7-8

**目标**：降容，优先主项恢复

**推荐安排**：
```
仅恢复日 LISS 20-30分钟
- 例如：减载周或测试周前，快走或游泳

避免：任何可能疲劳的训练
```

#### 减载周

**目标**：恢复，有氧可维持或略增

**推荐安排**：
```
LISS 30-45分钟，3-4天/周
- 例如：每天快走，促进全身恢复
```

---

### 有氧类型对肌肉冲击的分析

不同有氧类型对肌肉冲击程度不同，需根据训练日安排选择合适的有氧类型。

#### 有氧类型分类（按肌肉冲击程度）

| 有氧类型 | 肌肉冲击程度 | 适用场景 | 注意事项 |
|---------|------------|---------|---------|
| **快走** | 低（下肢轻度参与） | 所有训练日后、非训练日 | 最安全，几乎不干扰恢复 |
| **自行车** | 低-中（下肢中度参与） | 卧推日/轻量训练日后 | 避免深蹲/硬拉后使用（股四疲劳） |
| **椭圆机** | 低-中（全身轻度参与） | 所有训练日后、非训练日 | 最推荐的有氧类型（低冲击+全身） |
| **游泳** | 低（全身参与，但无冲击） | 所有训练日后、非训练日 | 最佳恢复性有氧（水中无重力） |
| **慢跑** | 中-高（下肢高度参与） | **仅非训练日**，且与腿部训练隔开 1 天 | 深蹲/硬拉后绝对避免（冲击膝关节+延缓恢复） |
| **划船机** | 中（全身中度参与） | 卧推日/轻量训练日后 | 对下背有要求，硬拉后需避免 |
| **HIIT（冲刺/波比跳）** | 高（全身高度参与） | **不建议**力量训练者 | 干扰恢复，影响力量表现 |

#### 不同训练日后的有氧类型推荐

| 训练日 | 推荐有氧类型 | 不推荐有氧类型 | 原因 |
|---------|---------|------------|------|
| **深蹲日** | 非训练日快走/游泳 | 自行车/慢跑/划船机 | 股四已疲劳，避免任何下肢主导的有氧 |
| **卧推日** | 自行车/椭圆机/快走 | 慢跑（如次日为深蹲/硬拉） | 上肢训练对腿部影响小，可做低冲击有氧 |
| **硬拉日** | 非训练日快走/游泳 | 自行车/划船机/慢跑 | 下背/后链已疲劳，避免任何后链主导的有氧 |
| **轻量训练日** | 椭圆机/自行车/快走/游泳 | HIIT | 疲劳度低，可承受中等强度有氧 |
| **减载周** | 所有低-中冲击有氧 | HIIT | 减载周以恢复为主，可做稍多的有氧 |

#### 单独有氧日的安排（如用户时间够多）

**适用用户**：每周可训练 4-5 天，希望增加有氧但又不干扰力量训练恢复。

**推荐安排**：
```
示例 1（4 天/周力量 + 1 天有氧）：
- 周一：深蹲
- 周二：卧推
- 周三：休息（或快走 30 分钟 LISS）
- 周四：硬拉
- 周五：上肢辅助
- 周六：休息
- 周日：单独有氧日（自行车 45 分钟 LISS 或游泳 30 分钟）

示例 2（3 天/周力量 + 2 天有氧）：
- 周一：深蹲 + 卧推
- 周二：休息（或快走 30 分钟 LISS）
- 周三：硬拉 + 上肢辅助
- 周四：休息
- 周五：核心 + 有氧（椭圆机 30 分钟 LISS）
- 周六：休息
- 周日：长距离快走/游泳 45 分钟 LISS
```

**优点**：
- 完全不干扰力量训练恢复
- 有氧质量更高（无疲劳叠加）
- 可承受更长时间的有氧（45-60 分钟）

**注意事项**：
- 单独有氧日也需遵循"低冲击优先"原则
- 有氧日后第二天如安排深蹲/硬拉，需降低有氧强度（或改为快走）

---


### 有氧训练的注意事项

#### 避免干扰力量恢复

**原则**：
1. **LISS > HIIT**：低强度稳态有氧不干扰力量恢复
2. **距离 > 强度**：快走 45 分钟 > 冲刺 20 分钟
3. **时机选择**：非训练日 > 训练后（避免叠加疲劳）

#### 有氧与 MRV 审计

**注意**：有氧训练**不占用**肌群 MRV 配额（不计入力量训练的组数）

**但需考虑**：
- 时间占用：有氧 30 分钟 + 力量训练 90 分钟 = 120 分钟（可能过长）
- 恢复占用：过度有氧 → 影响力量训练恢复

**建议**：
- 有氧安排在**非训练日**或**训练后低容量日**（如核心训练日）
- 如时间有限，优先保证力量训练

## AMRAP + RPE 估算 1RM（非竞技用户）

### 为什么不用 1RM 测试？

**适用用户**：
- 非竞技力量举运动员
- 不想在训练中冒险测试 1RM（受伤风险）
- 希望通过训练数据估算当前力量水平

**AMRAP + RPE 估算方法**：
- **AMRAP**：As Many Reps As Possible（尽可能多反复）
- **RPE**：自觉疲劳度评分
- **结合两者**：估算 1RM，误差约 10%

---

### 估算公式

#### 公式 1：基于 AMRAP 次数（Epley Formula）

```
1RM ≈ 重量 × (1 + 次数 / 30)
```

**示例**：
```
卧推：100kg × 5 次 AMRAP
1RM ≈ 100 × (1 + 5/30) = 100 × 1.167 = 116.7kg

实际 1RM：120kg
误差：≈ 2.8%（较准确）
```

#### 公式 2：基于 RPE（RPE to 1RM Table）

| RPE | 剩余次数（RIR） | 重量占 1RM 百分比 |
|-----|----------------|-------------------|
| 6 | 4 | ~80% |
| 7 | 3 | ~85% |
| 8 | 2 | ~90% |
| 8.5 | 1.5 | ~92.5% |
| 9 | 1 | ~95% |
| 9.5 | 0.5 | ~97.5% |
| 10 | 0 | 100% |

**示例**：
```
深蹲：140kg @ RPE 8（还能做 2 次）
1RM ≈ 140 / 0.9 = 155.6kg

实际 1RM：157.5kg
误差：≈ 1.2%（较准确）
```

#### 公式 3：AMRAP + RPE 结合（最准确）

**步骤**：
1. 做一组 AMRAP（例如 100kg × 8 次）
2. 记录最后 1 次的 RPE（例如 RPE 9）
3. 用 AMRAP 公式估算，再用 RPE 修正

**示例**：
```
卧推：100kg × 8 次 AMRAP，最后 1 次 RPE 9
AMRAP 估算：100 × (1 + 8/30) = 126.7kg
RPE 9 → 约 95% 1RM
修正：126.7 / 0.95 = 133.4kg

实际 1RM：130kg
误差：≈ 2.6%
```

---

### AMRAP + RPE 在周期中的安排

#### 容量期 W1-3

**目标**：不测试，仅记录训练数据

**安排**：
```
每次主项训练后记录：
- 重量
- 次数
- RPE

例如：
W1 深蹲：5×8 @ RPE 6, 90kg ×8 ×5
- 最后 1 组如果能做 10 次（RPE 6 → RIR 4）→ 1RM ≈ 90 / 0.8 = 112.5kg
```

#### 力量期 W5-6

**目标**：可选做一次 AMRAP 测试（非力竭）

**安排**：
```
W5 或 W6，选择一天做 AMRAP 测试：
- 选择次重重量（例如 85% 预计 1RM）
- 做 AMRAP（不超过 RPE 9）
- 用公式计算估算 1RM

例如：
W5 卧推：100kg AMRAP × 6 次，RPE 9
1RM ≈ 100 / 0.95 = 105.3kg（估算）
```

#### 冲刺期 W7-8

**目标**：不做 AMRAP（避免疲劳），仅用 RPE 监测

**安排**：
```
每次 TS 组记录 RPE：
- 如果 TS 重量 130kg @ RPE 8 → 1RM ≈ 130 / 0.9 = 144.4kg（估算）
- 如果 TS 重量 132.5kg @ RPE 8.5 → 1RM ≈ 132.5 / 0.925 = 143.2kg（估算）
```

#### 测试周（可选）

**目标**：如用户想要准确 1RM，可安排测试周

**安排**：
```
W9（测试周）：
- 周一：深蹲 1RM 测试
- 周三：卧推 1RM 测试
- 周五：硬拉 1RM 测试

或不测试，继续用 AMRAP + RPE 估算
```

### AMRAP + RPE 估算的误差分析

**误差来源**：

1. **技术波动**：AMRAP 时技术变形 → 次数偏少 → 低估 1RM
2. **RPE 误判**：新手常低估 RPE → 高估 1RM
3. **疲劳积累**：训练后期 AMRAP → 次数偏少 → 低估 1RM

**误差范围**：
- **有经验者**：±5-10%
- **新手**：±10-20%

**如何降低误差**：
1. **多次测量**：用不同重量/次数多次 AMRAP，取平均值
2. **RPE 校准**：新手先练习 RPE 判断（见 `references/autoregulation.md`）
3. **结合训练经验**：如果估算结果明显偏离训练日志趋势 → 手动修正

## 常见错误

### OHP

1. ❌ OHP 时腰背过度伸展 → ✅ 核心刚性，臀部夹紧
2. ❌ Push Press 代替 OHP → ✅ 两者互补，不能互相替代
3. ❌ OHP 重量过大（RPE 9+） → ✅ OHP 作为辅助时 RPE 7-8

### 核心训练

1. ❌ 核心训练做到力竭 → ✅ 核心训练 RPE 6-7，避免过度疲劳
2. ❌ 冲刺期核心训练不降容 → ✅ 冲刺期核心训练降容 50%
3. ❌ 忽略抗旋转/抗侧屈 → ✅ 核心训练要全面（抗伸展/抗旋转/抗侧屈）

### 有氧训练

1. ❌ 容量期做 HIIT → ✅ 只做 LISS（低强度稳态）
2. ❌ 有氧安排在深蹲/硬拉前一天 → ✅ 有氧与腿部训练隔开至少 1 天
3. ❌ 有氧时间过长（>60分钟） → ✅ 力量训练者有氧 20-45 分钟足够

### AMRAP + RPE 估算

1. ❌ AMRAP 做到力竭（RPE 10） → ✅ AMRAP 最多 RPE 9（保留 1 次）
2. ❌ 用单次 AMRAP 结果直接定 1RM → ✅ 多次测量取平均值
3. ❌ 新手直接用公式不修正 → ✅ 结合训练经验手动修正
