## 一、深蹲辅助动作

### 1. 暂停深蹲（Paused Squat）

**功能**：消除牵张反射，强化底部启动力量

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌、臀大肌、核心 |
| **训练价值** | 改善底部姿势控制，直接转化到比赛深蹲 |
| **安排建议** | 容量期：3×8 @ RPE 7（重量为深蹲主项的 70-80%）；力量期：可替代一次深蹲主项 |
| **限制** | 无特殊器械要求；需要较好的踝关节灵活性 |
| **适用人群** | 所有力量举运动员 |

---

### 2. 箱式深蹲（Box Squat）

**功能**：训练"坐到底再启动"的发力模式，改善底部启动

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 臀大肌、股四头肌（侧重臀部） |
| **训练价值** | 消除牵张反射，强化"从静止启动"的能力；箱式可安全练习高负荷启动 |
| **安排建议** | 容量期：3×8 @ RPE 7；可作为主项替代 |
| **限制** | ⚠️ 容易养成"砸坐"习惯，需控制下落速度 |
| **适用人群** | 底部启动无力者 |

---

### 3. 节奏深蹲（Tempo Squat，3秒下）

**功能**：增加离心阶段时长，强化股四头肌和控制力

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌（离心侧重） |
| **训练价值** | 增加肌肉损伤（肌肥大效果）；改善动作控制 |
| **安排建议** | 容量期：3×8 @ RPE 6-7；不适合力量期/冲刺期 |
| **限制** | 重量较轻（约 60-70% 1RM）；⚠️ 训练量感很强 |
| **适用人群** | 中途粘滞点（大腿平行附近）者 |

---

### 4. 安全杠深蹲（Safety Bar Squat）

**功能**：改变力矩，更侧重上背和核心稳定

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 上背、核心、股四头肌 |
| **训练价值** | 握力/手腕有伤时的深蹲替代；强化上背稳定性 |
| **安排建议** | 容量期：3×8 @ RPE 7；可作为主项替代 |
| **限制** | 🔧 需要安全杠（Specialty bar）；普通健身房可能没有 |
| **适用人群** | 手腕/肩有伤、上背薄弱者 |

---

### 5. 休息-暂停深蹲（Rest-Pause Squat）

**功能**：在 RPE 8-8.5 的重量下，通过短休息做更多次数，强化锁定

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 臀大肌、股四头肌（锁定阶段） |
| **训练价值** | 高密度刺激，强化"最后 5-10cm"锁定力量 |
| **安排建议** | 力量期/冲刺期：1×3 + 休息 15秒 × 3次 @ RPE 8.5 |
| **限制** | ⚠️ CNS 疲劳大；不适合容量期；不适合新手 |
| **适用人群** | 锁定无力者（高级训练者） |

---

### 6. 窄距深蹲（Close Stance Squat）

**功能**：改变力矩，更侧重股四头肌

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌（侧重） |
| **训练价值** | 补强股四头肌短板；改善膝盖上方粘滞点 |
| **安排建议** | 力量期：2×5 @ RPE 8；作为辅助 |
| **限制** | ⚠️ 髋部较窄者可能不适；需注意膝盖追踪 |
| **适用人群** | 股四头肌力量不足者 |

---

### 7. 泽奇深蹲（Zercher Squat）

**功能**：前蹲变式，肘部承托杠铃，强化上肢姿态和核心

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌、臀大肌、上背、二头肌 |
| **训练价值** | 强化上肢姿态（上背挺直）；直接转化到硬拉锁定 |
| **安排建议** | 力量期/冲刺期：2×5-8 @ RPE 8；作为辅助 |
| **限制** | 🔧 需要柔性杠铃或垫子保护肘部；⚠️ 二头肌有伤慎用 |
| **适用人群** | 锁定无力、上背薄弱者 |

---

### 8. 腿举（Leg Press）

**功能**：在稳定环境下高容量刺激股四头肌

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌（侧重）、臀大肌 |
| **训练价值** | 安全堆容量（无脊柱负荷）；适合容量期高次数 |
| **安排建议** | 容量期：3×10-12 @ RPE 7；作为辅助 |
| **限制** | 🔧 需要腿举机；转移到大项有限（无神经适应） |
| **适用人群** | 股四头肌力量不足者；下背有伤者的深蹲替代 |

---

### 9. 腿伸展（Leg Extension）

**功能**：孤立刺激股四头肌

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌（孤立） |
| **训练价值** | 肌肥大（容量期）；改善股四头肌分离度 |
| **安排建议** | 容量期：3×12-15 @ RPE 6-7；作为辅助收尾 |
| **限制** | 🔧 需要腿伸展机；⚠️ 膝盖有伤慎用 |
| **适用人群** | 股四头肌明显短板者 |

---

### 10. 保加利亚分腿蹲（Bulgarian Split Squat）

**功能**：单腿稳定性训练，改善左右不平衡

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌、臀大肌、核心稳定 |
| **训练价值** | 改善单腿稳定性；纠正左右不平衡 |
| **安排建议** | 容量期：3×8-10/腿 @ RPE 6-7；作为辅助 |
| **限制** | ⚠️ 平衡要求高；重量受限（哑铃/空杆） |
| **适用人群** | 有明显左右不平衡者；单侧薄弱者 |

---

### 11. 前蹲（Front Squat）

**功能**：杠铃置于前肩，垂直躯干，侧重股四头肌和核心

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌（侧重）、臀大肌、核心（抗屈曲） |
| **训练价值** | ✅ **最直接转化到深蹲核心力量**；改善躯干垂直姿势；对股四头肌刺激更大 |
| **安排建议** | 容量期：3×8 @ RPE 7；可作为主项替代（约占后蹲重量的70-80%） |
| **限制** | ⚠️ 需要足够的踝关节灵活性；手腕/肩部有伤慎用（可使用交叉握法或肩带辅助） |
| **适用人群** | 所有力量举运动员（尤其股四头肌薄弱者） |

## 二、卧推辅助动作

### 1. 暂停卧推（Paused Bench）

**功能**：消除牵张反射，强化底部控制

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 胸大肌、肱三头肌、前三角肌 |
| **训练价值** | ✅ **最直接转化到比赛卧推**；改善底部控制 |
| **安排建议** | 容量期：3×8 @ RPE 7（重量为卧推主项的 70-80%）；力量期：可替代一次卧推主项 |
| **限制** | 无特殊器械要求 |
| **适用人群** | 所有力量举运动员 |

---

### 2. Spoto 卧推（Spoto Press / 长杠程卧推）

**功能**：不触胸、悬空暂停 1-2 秒，强化全程控制

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 胸大肌（全程）、肱三头肌、前三角肌 |
| **训练价值** | 无反弹、强张力、控杆好；特别适合改善动作和突破粘滞点 |
| **安排建议** | 容量期：3×8 @ RPE 7；力量期：2×5 @ RPE 8 |
| **限制** | ⚠️ 对肩膀灵活性有一定要求；重量比常规卧推轻 10-15% |
| **适用人群** | 底部粘滞（杠铃离胸后前 5-10cm）者 |

> **与暂停卧推的区别**：暂停卧推是触胸暂停；Spoto 卧推是不触胸、在杠铃离胸 1-2cm 处暂停。

---

### 3. 宽距卧推（Wide Grip Bench）

**功能**：缩短杠铃行程，侧重胸肌刺激

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 胸大肌（侧重）、前三角肌 |
| **训练价值** | 缩短行程，改善"底部粘滞"；胸肌主导的卧推变式 |
| **安排建议** | 容量期：3×8 @ RPE 7；作为辅助 |
| **限制** | ⚠️ 肩膀有伤慎用（肩峰撞击风险）；握距不能超过肩宽 1.5 倍（规则限制） |
| **适用人群** | 底部粘滞（胸力量不足）者 |

---

### 4. 窄距卧推（Close Grip Bench）

**功能**：缩短杠铃行程，侧重肱三头肌

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 肱三头肌（侧重）、胸大肌（内侧） |
| **训练价值** | 改善中途粘滞（肘部 45° 附近）；直接强化肱三头肌 |
| **安排建议** | 容量期：3×8 @ RPE 7；力量期：2×5 @ RPE 8 |
| **限制** | ⚠️ 手腕有伤慎用（窄握对腕部压力大） |
| **适用人群** | 中途粘滞（三头力量不足）者 |

---

### 5. 地板卧推（Floor Press）

**功能**：躺在地板上，行程缩短（大臂触地即止），强化顶部锁定

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 肱三头肌（侧重）、胸大肌（上部） |
| **训练价值** | 行程缩短 → 强迫"顶部锁定"力量；安全性高（无杠铃砸胸风险） |
| **安排建议** | 力量期：2×5-6 @ RPE 8；作为辅助 |
| **限制** | 🔧 需要空地或平板；行程过短（对胸肌刺激有限）；⚠️ 不适合容量期高次数 |
| **适用人群** | 锁定无力（三头力量不足）者；居家/无卧推架时的替代 |

> **与常规卧推的区别**：地板卧推大臂下落到触地即止，行程比常规卧推短 5-10cm，更侧重三头肌和锁定力量。

---

### 6. JM 推举（JM Press）

**功能**：卧推变式，强调肘部伸展力量

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 肱三头肌（侧重）、胸大肌 |
| **训练价值** | 改善中途粘滞（三头力量不足）；肘部伸展的专项训练 |

| **限制** | ⚠️  elbow flare 需控制（避免肘部外展过大）；技术难度中等 |
| **适用人群** | 中途粘滞（三头力量不足）者 |

## 三、硬拉辅助动作

### 1. 暂停硬拉（Paused Deadlift）

**功能**：在膝盖下方或杠铃离地 5cm 处暂停，强化启动力量

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 臀大肌、腘绳肌、股四头肌（启动阶段） |
| **训练价值** | 消除牵张反射，纯启动力量输出；改善"离地困难" |
| **安排建议** | 容量期：3×6 @ RPE 7（重量为硬拉主项的 70-80%） |
| **限制** | ⚠️ 腰椎压力增加，需谨慎；不建议冲刺期使用 |
| **适用人群** | 启动无力者 |

---

### 2. Deficit 硬拉（站在 2-3cm 平台上）

**功能**：站在小平台上，增加 ROM（活动范围），强化启动力量

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌（侧重）、臀大肌、腘绳肌 |
| **训练价值** | 增加股四头肌参与；改善"离地困难"；更接近深蹲的启动模式 |
| **安排建议** | 容量期：3×6 @ RPE 7；作为辅助 |
| **限制** | ⚠️ 腰椎压力增加（ROM 增加）；踝关节灵活性要求高；不建议冲刺期使用 |
| **适用人群** | 启动无力者（股四头肌力量不足） |

---

### 3. 六角杠硬拉（Hex Bar / Trap Bar Deadlift）

**功能**：站在六角杠内，对脊柱更友好的硬拉变式

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 股四头肌、臀大肌、腘绳肌 |
| **训练价值** | 脊柱垂直受力（减少下背压力）；适合堆容量；可作为硬拉主项替代 |
| **安排建议** | 容量期：3×8 @ RPE 7；作为辅助或主项替代 |
| **限制** | 🔧 需要六角杠（Specialty bar）；转移到传统硬拉有限 |
| **适用人群** | 下背有伤者的硬拉替代；新手硬拉入门 |

---

### 4. 节奏硬拉（Tempo Deadlift，3秒拉起）

**功能**：增加离心/向心阶段时长，强化控制和后链力量

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 臀大肌、腘绳肌、下背（竖脊肌） |
| **训练价值** | 增加肌肉损伤（肌肥大）；改善"膝盖上方粘滞" |
| **安排建议** | 力量期：2×5 @ RPE 8；作为辅助 |
| **限制** | ⚠️ 重量较轻；不适合冲刺期 |
| **适用人群** | 膝盖上方粘滞者 |

---

### 5. Block 硬拉（从膝盖高度启动）

**功能**：杠铃置于 Block（2-3 块）上，从膝盖高度启动，强化中途/锁定力量

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 臀大肌、腘绳肌、背部（锁定阶段） |
| **训练价值** | 专项训练"过膝后"阶段；可直接用比赛重量练习锁定 |
| **安排建议** | 力量期/冲刺期：2×3 @ RPE 8.5 |
| **限制** | 🔧 需要 Block（或杠铃片垫高）；⚠️ 起始位置对下背压力大 |
| **适用人群** | 膝盖上方粘滞/锁定无力者 |

---

### 6. 罗马尼亚硬拉（RDL / Romanian Deadlift）

**功能**：从站立位开始，髋铰链主导，强化后链

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 腘绳肌（侧重）、臀大肌、下背（竖脊肌） |
| **训练价值** | ✅ **最直接强化后链**；改善硬拉"锁定阶段"；对下背耐力有很大帮助 |
| **安排建议** | 容量期：3×8 @ RPE 7；作为辅助 |
| **限制** | ⚠️ 下背有伤慎用（可改用 Good Morning 轻重量） |
| **适用人群** | 所有力量举运动员 |

---

### 7. Good Morning（早安式）

**功能**：杠铃置于上背，髋铰链主导，强化后链和下背

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 腘绳肌、臀大肌、下背（竖脊肌） |
| **训练价值** | 改善硬拉"锁定阶段"；强化核心抗屈曲能力 |
| **安排建议** | 容量期：3×8 @ RPE 7；作为辅助 |
| **限制** | ⚠️ 下背压力较大；重量不宜过大；新手建议用轻杆或 PVC 管练习 |
| **适用人群** | 锁定无力者；下背耐力不足者 |

---

### 8. 臀推（Hip Thrust）

**功能**：仰卧，上背贴凳，杠铃置于髋部，髋伸展主导

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 臀大肌（侧重）、腘绳肌 |
| **训练价值** | ✅ **最直接孤立刺激臀大肌**；对硬拉锁定阶段有直接帮助；女性训练者尤其喜欢 |
| **安排建议** | 容量期：3×10-12 @ RPE 7；作为辅助 |
| **限制** | 🔧 需要深蹲架/力量架 + 长凳；⚠️ 髋部不适者慎用 |
| **适用人群** | 臀大肌力量不足者；女性训练者 |

## 四、背部辅助动作

### 1. Pendlay 划船（Pendlay Row）

**功能**：杠铃置于地面，每次从静止启动，爆发力拉至腹部

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 上背（ rhomboids、trapezius）、二头肌 |
| **训练价值** | 强化上背爆发力（直接转移到硬拉锁定）；改善俯身姿势下背部耐力 |
| **安排建议** | 容量期：3×8-10 @ RPE 7（重量为体重的 40-50%）；力量期：2×6 @ RPE 8 |
| **限制** | ⚠️ 下背有伤慎用（可改用坐姿划船）；重量不宜过大（技术 > 重量） |
| **适用人群** | 有一定训练经验者 |

---

### 2. 海豹划船机（Seal Row Machine）

**功能**：俯卧在专用器械上，躯干/大腿贴靠垫，孤立上背

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 上背（rhomboids、trapezius、后三角肌） |
| **训练价值** | 消除下背参与 → 适合下背有伤或疲劳时替代自由重量划船 |
| **安排建议** | 容量期：3×10-12 @ RPE 7-8；力量期：2×8 @ RPE 8；冲刺期：2×6 @ RPE 8 |
| **限制** | 🔧 需要海豹划船机（非所有健身房都有） |
| **适用人群** | 新手；下背有伤/疲劳者 |

---

### 3. 俯身哑铃划船（Bent-over Dumbbell Row）

**功能**：单臂/双臂俯身，哑铃划船，双侧独立训练

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 背阔肌、菱形肌、二头肌 |
| **训练价值** | 双侧独立训练（纠正左右不平衡）；适合新手建立背部收缩感觉 |
| **安排建议** | 容量期：3×10 @ RPE 7 |
| **限制** | ⚠️ 平衡要求（俯身姿势）；重量受限（哑铃最大重量） |
| **适用人群** | 新手；有明显左右不平衡者 |

---

### 4. 坐姿器械划船（Seated Cable Row）

**功能**：坐姿，器械固定轨迹，水平拉

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 上背（水平拉）、菱形肌 |
| **训练价值** | 动作固定，适合孤立；可作为"保底"背部动作 |
| **安排建议** | 容量期：3×10-12 @ RPE 7 |
| **限制** | 🔧 需要坐姿划船机；转移到大项有限（轨迹固定） |
| **适用人群** | 新手 |

---

### 5. 引体向上 / 辅助引体（Pull-up / Assisted Pull-up）

**功能**：垂直拉，训练背阔肌

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 背阔肌、二头肌 |
| **训练价值** | 与划船（水平拉）互补；改善背部宽度 |
| **安排建议** | 每次训练后：3×6-10 @ RPE 6-7 |
| **限制** | ⚠️ 体重较大者可能做不了（可用辅助引体机） |
| **适用人群** | 所有训练者 |

## 五、核心与全身辅助动作

### 1. 北欧挺（Nordic Curl / 腘绳肌弯举）

**功能**：跪姿，身体前倾，腘绳肌离心收缩控制下落

| 项目 | 内容 |
|------|------|
| **主要刺激肌群** | 腘绳肌（离心侧重） |
| **训练价值** | ✅ **最有效的腘绳肌离心训练**；预防腘绳肌拉伤；改善硬拉/深蹲后链力量 |
| **安排建议** | 容量期：3×5-8 @ RPE 6-7（离心控制） |
| **限制** | 🔧 需要北欧挺专用器械或协助者（固定脚踝）；⚠️ 腘绳肌有伤慎用；技术难度高 |
| **适用人群** | 中高级训练者；腘绳肌薄弱者 |

### 2. 逆向北欧挺（Reverse Nordic Curl）

**功能**：高跪姿，身体向后倒，股四头肌离心收缩控制下落

| **安排建议** | 容量期：3×5-8 @ RPE 6-7（离心控制） |
| **限制** | ⚠️ 对股四头肌伸展性要求较高；如果无法完成，不一定是因为力量差，可能是柔韧性不足 |

## 八、动作选择决策树

### 深蹲薄弱点 → 辅助动作选择

```
底部启动无力
├── 暂停深蹲 ✅（首选）
├── 前蹲
└── 箱式深蹲

中途粘滞（大腿平行附近）
├── 节奏深蹲（3秒下）
└── 安全杠深蹲

锁定无力（最后 5-10cm）
├── 休息-暂停深蹲
├── 窄距深蹲
└── 泽奇深蹲

股四头肌力量不足
├── 前蹲 ✅（首选，核心转化度高）
├── 腿举
├── 腿伸展
└── 保加利亚分腿蹲
```

### 卧推薄弱点 → 辅助动作选择

```
上胸薄弱
├── 哑铃上斜卧推 ✅（首选）
└── 杠铃上斜卧推

底部粘滞（杠铃离胸后前 5-10cm）
├── 暂停卧推 ✅（首选）
├── Spoto 卧推
└── 宽距卧推

中途粘滞（肘部 45° 附近）
├── 窄距卧推 ✅（首选）
├── 地板卧推
└── JM 推举

锁定无力（最后 5-10cm）
├── 板卧推
└── 链/弹力带卧推

下胸/三头薄弱
├── 双杠臂屈伸 ✅（首选）
└── 夹胸/飞鸟

肩部稳定性差
├── 面拉 ✅（每次训练后）
├── 哑铃肩推
├── 杠铃肩推
├── 侧平举
├── 反向飞鸟
└── 哑铃飞鸟
```

### 硬拉薄弱点 → 辅助动作选择

```
启动无力（离地困难）
├── 暂停硬拉 ✅（首选）
├── Deficit 硬拉
└── 六角杠硬拉

膝盖上方粘滞
├── 节奏硬拉
└── Block 硬拉

锁定无力（最后 5-10cm）
├── 罗马尼亚硬拉（RDL）✅（首选）
├── Good Morning
└── 臀推

握力不足
├── 农夫行走 ✅（首选）
├── 握力器训练
└── 正反握/Hook 握
```
