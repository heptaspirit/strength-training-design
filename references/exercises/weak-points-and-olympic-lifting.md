## 三大项薄弱点识别与辅助动作选择

### 用户薄弱点自我评估引导

**适用场景**：用户是新手/小白，不知道自己的薄弱点在哪里。

#### 引导流程

**第一步：礼貌询问**

```
"为了更好地设计适合你的训练计划，我需要了解一下你在深蹲/卧推/硬拉三个动作中是否有感觉特别困难的部分。

如果你不确定自己的薄弱点，我们可以简单聊聊：
1. 你通常在动作的哪个位置感觉最吃力？（例如：深蹲底部启动 / 卧推杠铃离胸后的前 10cm / 硬拉刚离地时）
2. 你最近一次训练这三个动作时，哪个阶段让你感觉最累或最不稳定？
3. 你是否曾经在训练中因为某个位置卡住而失败过？

如果你暂时不想分析薄弱点，我也可以按照通用方案为你安排辅助动作，后续训练中再慢慢调整。你觉得怎么样？"
```

**第二步：根据用户回答判断**

| 用户回答 | 处理方式 |
|---------|---------|
| "我不知道/我不懂" | 按通用辅助动作安排（见下方"通用辅助动作方案"） |
| "我某个位置感觉吃力" | 引导用户更详细描述（见下方"详细引导问题"） |
| "我卡在某某位置失败过" | 直接对应到薄弱点表，推荐辅助动作 |
| "我想先做通用方案，后续再调整" | 按通用辅助动作安排，并记录"待后续调整" |

**第三步：提供通用辅助动作方案（用户不想分析薄弱点时）**

| 主项 | 通用辅助动作（适合大多数人） | 训练安排 |
|------|--------------------------|---------|
| **深蹲** | 暂停深蹲（Paused Squat）<br>泽奇深蹲（Zercher Squat）<br>腿举（Leg Press） | 容量期：3×8 @ RPE 7<br>力量期：2×5 @ RPE 8 |
| **卧推** | 暂停卧推（Paused Bench）<br>窄距卧推（Close Grip Bench） | 容量期：3×8 @ RPE 7<br>力量期：2×5 @ RPE 8 |
| **硬拉** | 罗马尼亚硬拉（RDL）<br>Deficit 硬拉（站在 2-3cm 平台上） | 容量期：3×6 @ RPE 7<br>力量期：2×5 @ RPE 8 |

**第四步：详细引导问题（用户愿意分析但不确定时）**

**深蹲引导问题**：
```
1. 你深蹲时，是从底部启动最吃力，还是大腿平行位置卡住，还是最后站直时锁定无力？
2. 你是否能轻松下蹲到大腿平行或略低于平行的位置？还是感觉底部不稳定？
3. 你深蹲时，是股四头肌（大腿前侧）先累，还是臀/腘旁肌（大腿后侧）先累？
```

**卧推引导问题**：
```
1. 你卧推时，是杠铃刚离胸的前 10cm 最吃力，还是肘部弯曲 45° 附近卡住，还是最后锁定无力？
2. 你卧推时，是胸部先累，还是三头肌（大臂后侧）先累？
3. 你卧推时，杠铃是否容易摇晃？肩部是否感觉不稳定？
```

**硬拉引导问题**：
```
1. 你硬拉时，是杠铃刚离地最吃力，还是过膝后卡住，还是最后站立锁定无力？
2. 你硬拉时，是下背先累，还是臀/腘旁肌（大腿后侧）先累，还是握力先不够？
3. 你硬拉启动时，是否感觉杠铃"粘在地上"起不来？
```

**第五步：根据引导结果推荐辅助动作**

将用户回答对应到"常见薄弱点与辅助动作表"（见下方各动作详细分析），选择 1-2 个最关键的动作加入计划。

#### 通用辅助动作方案（默认）

如果用户不愿意分析薄弱点，或确实是纯新手，使用以下**通用方案**：

```
周一（深蹲日）：
- 主项：深蹲
- 辅助：暂停深蹲 3×8 @ RPE 7（改善底部控制）

周三（卧推日）：
- 主项：卧推
- 辅助：暂停卧推 3×8 @ RPE 7（改善底部力量）

周五（硬拉日）：
- 主项：硬拉
- 辅助：罗马尼亚硬拉（RDL）3×8 @ RPE 7（改善后链力量）
```

**后续调整**：
- 训练 4-6 周后，根据用户反馈（哪个位置还是卡住）调整辅助动作
- 或要求用户提供训练视频，帮助识别薄弱点

---

### 为什么需要薄弱点分析？

**目的**：识别主项（深蹲/卧推/硬拉）的 sticking points（粘滞点），选择针对性辅助动作，突破力量瓶颈。

**原理**：
- 每个动作的力速曲线（Force-Velocity Curve）不同，薄弱点反映特定关节角度/肌群的力量短板
- 针对性辅助动作 → 补强弱点 → 主项进步

## 深蹲（Squat）薄弱点分析

### 常见薄弱点与辅助动作

| 薄弱点位置 | 可能原因 | 推荐辅助动作 | 训练安排 |
|-----------|---------|-------------|---------|
| **底部启动无力**（离地后前 5-10cm） | 起始姿势不佳 / 髋/膝伸展力量不足 | **暂停深蹲**（Paused Squat）<br>**箱式深蹲**（Box Squat） | 容量期：3×8 @ RPE 7<br>力量期：2×5 @ RPE 8 |
| **中途粘滞点**（大腿平行附近） | 股四头肌力量不足 / 节奏不当 | **节奏深蹲**（3秒下）<br>**安全杠深蹲**（Safety Bar Squat） | 容量期：3×8 @ RPE 7<br>可作为主项替代 |
| **锁定无力**（站立最后 5-10cm） | 髋伸展不足 / 核心不稳定 / 核心抗屈曲不足 | **休息-暂停深蹲**（Rest-Pause）<br>**窄距深蹲**（Close Stance）<br>**泽奇深蹲**（Zercher Squat） | 力量期/冲刺期：2×3 @ RPE 8.5 |
| **股四头肌力量不足**（整体无力） | 股四发展滞后 | **腿举**（Leg Press）<br>**腿伸展**（Leg Extension）<br>**保加利亚分腿蹲** | 容量期：3×10-12 @ RPE 7<br>作为辅助，非主项 |

### 暂停深蹲（Paused Squat）技术要点

**动作描述**：
- 下蹲至大腿平行或略低于平行
- **暂停 2-3 秒**（底部无反弹）
- 爆发性站起

**训练价值**：
- 消除牵张反射（Stretch Reflex）→ 纯力量输出
- 改善底部姿势控制
- 直接转化到比赛深蹲

**安排建议**：
- **容量期**：作为辅助动作，3×8 @ RPE 7（重量为深蹲主项的 70-80%）
- **力量期**：可替代一次深蹲主项（如周五深蹲改为暂停深蹲）

## 背部辅助动作库（划船变式与器械）

### 为什么背部训练重要？

**背部力量直接影响三大项表现**：
- **深蹲**：上背稳定性决定杠铃位置（低杠深蹲尤其依赖）
- **卧推**：肩胛骨后收与稳定是力量的基础（上背=卧推的"根基"）
- **硬拉**：后链（臀/腘旁/下背）是硬拉的主驱动肌群

**训练原则**：
- 背部训练分两类：**垂直拉**（引体、滑轮下拉）+ **水平拉**（各类划船）
- 力量举计划中应**优先水平拉**（更直接转移到大项）
- 每次训练后以**2-3个背部动作**收尾（容量期可更多）

---

### 划船变式详解

#### Pendlay 划船（Pendlay Row / 爆发力划船）

**动作描述**：
- 杠铃置于地面（每次动作从完全静止启动）
- 爆发力将杠铃拉至下腹部/肚脐位置
- 缓慢放下（离心控制 2-3 秒），杠铃触地后再进行下一次

**技术要点**：
- 躯干与地面平行（严格俯身，不借用下肢蹬地）
- 拉起时肩胛骨主动后收，肘部主导（非二头肌）
- 每次从地面重启 → 消除牵张反射，纯爆发力输出

**训练价值**：
- 强化上背爆发力（直接转移到硬拉锁定）
- 改善俯身姿势下背部耐力（硬拉后链稳定）
- 对下背等长收缩能力有较高要求（核心稳定）

**安排建议**：
- **容量期**：3×8-10 @ RPE 7（重量为体重的 40-50%）
- **力量期**：2×6 @ RPE 8
- **冲刺期**：不建议（下背疲劳影响硬拉恢复）
- 安排在深蹲日或硬拉日后（非卧推日前，避免影响肩胛骨稳定）

**注意事项**：
- ⚠️ 下背有伤慎用（可改用坐姿划船替代）
- 重量不宜过大（爆发力动作，技术 > 重量）

---

#### 海豹划船机（Seal Row Machine）

**动作描述**：
- 俯卧在专用器械上（躯干/大腿贴靠垫，双脚固定）
- 双手持握把，将负重拉向肚脐/下腹部位置
- 动作轨迹固定，无需担心平衡

**训练价值**：
- **孤立上背部**（胸棘肌、菱形肌、后三角肌）
- 消除下背参与 → 适合下背有伤或疲劳时替代自由重量划船
- 动作轨迹固定 → 适合新手建立背部收缩感觉

**与自由重量划船对比**：

| 维度 | Pendlay 划船 | 海豹划船机 |
|------|--------------|------------|
| 核心稳定需求 | 高（俯身对抗重力） | 低（器械支撑） |
| 下背参与 | 高（等长收缩） | 无（孤立） |
| 重量负荷 | 中等（40-50% 体重） | 可较重（机械优势） |
| 适合人群 | 有经验者 | 新手 / 下背有伤 | 

**安排建议**：
- **容量期**：3×10-12 @ RPE 7-8（可较重）
- **力量期**：2×8 @ RPE 8
- **冲刺期**：2×6 @ RPE 8（保持背部张力，不影响主项恢复）
- 可安排在**任何训练日**（对主项干扰小）

**注意事项**：
- 动作顶端停顿 1 秒（挤压肩胛骨），强化收缩感觉
- 不要借助惯性（控制离心 2-3 秒）

---

#### 其他背部辅助动作（备选库）

| 动作 | 主要刺激肌群 | 安排建议 | 备注 |
|------|------------|---------|------|
| **俯身哑铃划船** | 背阔肌、菱形肌 | 容量期：3×10 @ RPE 7 | 适合新手，双侧独立训练 |
| **坐姿器械划船** | 上背（水平拉） | 容量期：3×10-12 @ RPE 7 | 动作固定，适合孤立 |
| **引体向上 / 辅助引体** | 背阔肌、二头肌 | 每次训练后：3×6-10 @ RPE 6-7 | 垂直拉，与划船互补 |
| **面拉（Face Pull）** | 后三角肌、肩袖 | 每次训练后：3×15 @ RPE 6 | 肩部健康，卧推必备 |
| **反向飞鸟（Reverse Fly）** | 后三角肌、菱形肌 | 3×15 @ RPE 6-7 | 肩部稳定性 |

---

### 背部训练的 MRV 审计

**上背/背部容量审计示例**：
```
用户：男性，训练3年
安排：
- Pendlay 划船：3组（水平拉）
- 引体：3组（垂直拉）
- 面拉：3组（后三角/肩袖）
- 总计背部容量：9组/周
- 背部 MRV ≈ 20 组/周 ✅ 通过（9 < 20）
```

**调整原则**：
- 背部 MRV 较高（≈ 20 组/周），一般不易超标
- 若下背参与较多（Pendlay 划船、RDL），需注意与硬拉容量的叠加

## 卧推（Bench Press）薄弱点分析

### 常见薄弱点与辅助动作

| 薄弱点位置 | 可能原因 | 推荐辅助动作 | 训练安排 |
|-----------|---------|-------------|---------|
| **底部粘滞**（杠铃离胸后前 5-10cm） | 胸大肌力量不足 / 起始姿势不佳 | **暂停卧推**（Paused Bench）<br>**长杠程卧推**（Spoto Press）<br>**宽距卧推** | 容量期：3×8 @ RPE 7<br>力量期：2×5 @ RPE 8 |
| **中途粘滞点**（肘部 45° 附近） | 三头肌力量不足 /  elbow flare | **窄距卧推**（Close Grip Bench）<br>**地板卧推**（Floor Press）<br>**JM 推举**（JM Press） | 容量期：3×8 @ RPE 7<br>作为辅助 |
| **锁定无力**（肘部伸展最后 5-10cm） | 三头肌峰值收缩不足 | **板卧推**（Board Press）<br>**链/弹力带卧推**（Accommodating Resistance） | 力量期/冲刺期：2×3 @ RPE 8.5 |
| **肩部稳定性差**（杠铃摇晃） | 肩袖肌群不足 / 肩胛骨控制差 | **面拉**（Face Pull）<br>**反向飞鸟**（Reverse Fly）<br>**帕夫罗推**（Pavlo Press） | 每次训练后：3×15 @ RPE 6-7 |

### 暂停卧推（Paused Bench）技术要点

**动作描述**：
- 杠铃轻触胸部（或离胸 1-2cm）
- **暂停 1-2 秒**（无反弹）
- 爆发性推起

**训练价值**：
- 消除牵张反射 → 纯胸/三头力量输出
- 改善底部控制（比赛规则要求）
- **最直接转化到比赛卧推**

**安排建议**：
- **容量期**：作为辅助动作，3×8 @ RPE 7（重量为卧推主项的 70-80%）
- **力量期/冲刺期**：可替代一次卧推主项（如周五卧推改为暂停卧推）

**用户示例**（boyang）：
- 卧推底部粘滞 → 加入暂停卧推作为辅助
- 安排：W1-3: 3×8 @ RPE 7; W5-6: 2×5 @ RPE 8

## 硬拉（Deadlift）薄弱点分析

### 常见薄弱点与辅助动作

| 薄弱点位置 | 可能原因 | 推荐辅助动作 | 训练安排 |
|-----------|---------|-------------|---------|
| **启动无力**（离地困难） | 起始姿势不佳 / 膝/髋伸展不足 | **暂停硬拉**（Paused Deadlift）<br>**Deficit 硬拉**（站在 2-3cm 平台上）<br>**六角杠硬拉**（Hex Bar） | 容量期：3×6 @ RPE 7<br>作为辅助 |
| **膝盖上方粘滞**（杠铃过膝后卡住） | 髋铰链（Hip Hinge）不足 / 后链弱 | **节奏硬拉**（3秒拉起）<br>**Block 硬拉**（从膝盖高度启动） | 力量期：2×5 @ RPE 8 |
| **锁定无力**（完全站立最后 5-10cm） | 臀/髋伸展不足 / 骨盆前倾 | **罗马尼亚硬拉**（RDL）<br>**好 Morning**（Good Morning） | 容量期：3×8 @ RPE 7<br>作为辅助 |
| **握力不足**（杠铃滑落） | 握力滞后 | **农夫行走**（Farmer's Walk）<br>**握力器训练**<br>**正反握/ hook 握** | 每次训练后：3×10 秒握力器 |

### Deficit 硬拉技术要点

**动作描述**：
- 站在 2-3cm 平台上（增加 ROM）
- 从更低位置启动 → 强化起始力量
- **注意**：腰椎压力增加，需谨慎

**训练价值**：
- 改善启动力量（最弱环节）
- 增加股四参与（更接近深蹲）
- 强化核心稳定性

**安排建议**：
- **容量期**：作为辅助动作，3×6 @ RPE 7（重量为硬拉主项的 70-80%）
- **不建议**：冲刺期使用（下背疲劳叠加）

## 奥林匹克举重辅助（高翻/抓举）

### 为什么在力量举计划中加入奥举？

**价值**：
1. **全身爆发力**：高翻（Power Clean）训练全身协调发力
2. **神经适应**：快速动作改善高阈值运动单位招募
3. **技术转移**：髋/膝伸展力量直接转化到深蹲/硬拉
4. **多样性**：避免训练单调，降低过度使用伤风险

**适用人群**：
- 有奥举经验或愿意学习技术
- 希望提升全身爆发力（非纯力量举竞技）
- 生活/训练压力适中（奥举需要专注和技术练习）

---

### 高翻（Power Clean）安排

#### 技术要点

**动作阶段**：
1. **起始**：髋部高于膝盖，背部平直
2. **第一阶段拉**：腿+髋伸展，杠铃贴小腿上移
3. **爆炸阶段**：髋完全伸展，肩耸起，肘快速向前翻转
4. **接铃**：髋/膝快速弯曲（吸收冲击），杠铃落于锁骨/前肩
5. **恢复**：站直，完成

**关键提示**：
- **速度 > 重量**：高翻的价值在「爆炸力」，不在「最大重量」
- **技术优先**：重量轻但技术正确 > 重量重但技术崩坏
- **录像复盘**：每次高翻日建议录像，检查技术

#### 在周期中的安排

| 周期阶段 | 组次配置 | RPE/强度 | 目的 |
|---------|---------|---------|------|
| **容量期 W1-3** | 3×5 | RPE 7 | 技术练习 + 容量积累 |
| **减载周 W4** | 2×3 | RPE 6 | 恢复，保持技术感觉 |
| **力量期 W5-6** | 1×3-5 | RPE 7-8 | 保持爆发力，不堆容量 |
| **冲刺期 W7-8** | 1×2-3 | RPE 6-7 | 降容保技术，优先主项恢复 |

#### 与主项的搭配

**推荐搭配**（避免疲劳叠加）：
- ✅ **深蹲日 + 高翻**：前蹲/后蹲与高翻有技术转移
- ⚠️ **硬拉日 + 高翻**：下背疲劳叠加，谨慎安排（可降低硬拉容量）
- ✅ **卧推日 + 高翻**：无冲突

**示例训练日**：
```
周一：深蹲 + 高翻 + 上身拉
- 深蹲主项：5×8 @ RPE 6
- 高翻：3×5 @ RPE 7（技术优先）
- 引体：3×8 @ RPE 6
- 面拉：3×15 @ RPE 6
```

---

### 抓举（Snatch）与高抓（Power Snatch）

#### 抓举（Snatch）

**特点**：
- 全身协调性要求最高
- 技术难度极大（需要长期练习）
- 对力量举的直接转移有限

**建议**：仅适用于有经验者，或作为「技术日」主项（非力量举辅助）

#### 高抓（Power Snatch）

**特点**：
- 抓举的简化版（接铃在髋/膝高度，不需深蹲）
- 更适合作为力量举辅助
- 训练全身爆发力和协调性

**安排建议**：
- **容量期**：2×5 @ RPE 7（技术练习）
- **不建议**：力量期/冲刺期使用（技术难度高，干扰恢复）

---

### Clean Pull / Snatch Pull（拉举）

#### Clean Pull（高翻拉）

**动作描述**：
- 高翻的「拉起阶段」，不做翻站/接铃
- 专注髋/膝伸展力量

**价值**：
- 直接强化后链（臀/腘旁/下背）
- 技术难度低，适合大多数力量举运动员
- **最直接转化到硬拉**

**安排建议**：
- **容量期**：作为硬拉辅助，3×6 @ RPE 7-8
- **力量期/冲刺期**：2×5 @ RPE 8

#### Snatch Pull（抓举拉）

**动作描述**：
- 抓举的「拉起阶段」，不做抓举接铃
- 更强调上半身姿势控制

**价值**：
- 改善肩胛骨稳定性和肩袖控制
- 对上背力量有帮助

**安排建议**：
- **容量期**：作为辅助，2×6 @ RPE 7
- 可作为「动态日」主项（替代高翻）

---

### 膝上高翻/膝上高拉作为硬拉日热身/启动训练

#### 什么是膝上高翻/膝上高拉？

- **膝上高翻（Hang Power Clean）**：从膝上/大腿中段位置启动的高翻，接铃在锁骨/前肩位置（不需深蹲）
- **膝上高拉（Hang Clean Pull）**：只做高翻的"拉起阶段"，拉到肩部高度但不翻不接杠，技术难度更低，也被称作"高拉不翻"

#### 为什么作为硬拉日第一个动作有用？

**理论依据**：
1. **后链激活**：膝上变式通过爆发性髋/膝伸展，有效激活臀大肌、腘旁肌、竖脊肌（硬拉主驱动肌群）
2. **神经启动**：爆发性动作能快速激活高阈值运动单位，为后续大重量硬拉"预热"神经系统
3. **规避地面启动**：对地面启动困难（stick point 在离地后5-10cm）的训练者，膝上变式可以绕过最弱点，专注于"髋伸展爆发力"
4. **脊柱负荷较低**：膝上启动的 ROM 比地面硬拉短，cumulative spinal loading 较低，适合作为硬拉日前几个热身组后的"渐进加载"
5. **技术练习**：如果训练者本身练高翻，膝上高翻是标准高翻的技术阶梯（先练膝上，再练地面）

**适用场景**：
- ✅ 硬拉日第一个动作，作为"后链激活 + 神经启动"，替代部分传统热身
- ✅ 训练者地面硬拉启动困难（stick point 在离地后）
- ✅ 训练者希望减少硬拉从地面启动的 cumulative fatigue
- ✅ 训练者本身有奥举训练需求（高翻/高抓）

**不适用场景**：
- ❌ 奥举技术为零且不愿学习（可改用 RDL 或 Block 硬拉）
- ❌ 下背有伤（膝上高翻/高拉仍需脊柱支撑，可改用器械后链动作）
- ❌ 硬拉日是高频训练日（如每周2次硬拉），奥举动作可能干扰恢复

#### 安排建议

| 周期阶段 | 动作选择 | 组次配置 | RPE/强度 | 目的 |
|---------|---------|---------|----------|------|
| **容量期 W1-3** | 膝上高拉（Hang Clean Pull） | 3×5 | RPE 7 | 后链激活 + 技术练习 |
| **减载周 W4** | 膝上高拉 | 2×3 | RPE 6 | 保持技术感觉 |
| **力量期 W5-6** | 膝上高翻（Hang Power Clean） | 1×3-5 | RPE 7-8 | 爆发力保持，不堆容量 |
| **冲刺期 W7-8** | 膝上高拉（轻） | 1×2-3 | RPE 6-7 | 神经激活，不干扰恢复 |

**与主项搭配**：
- **硬拉日 + 膝上高翻/高拉**：作为第一个动作（替代部分传统热身），5-10 分钟后直接进入硬拉主项
- **重量选择**：膝上高翻/高拉的重量应为硬拉 1RM 的 40-60%（技术 > 重量）
- **组数控制**：硬拉日加入奥举动作后，需减少其他后链辅助动作组数（避免超过后链 MRV）

**示例硬拉日安排**：
```
周五：硬拉日
- 膝上高拉：3×5 @ RPE 7（硬拉 1RM 的 50%）
- 硬拉主项：4×6 @ RPE 6-7
- RDL（辅助）：2×8 @ RPE 7（若疲劳则取消）
- 划船（背部）：3×8 @ RPE 7
```

#### 注意事项

1. **技术优先**：膝上高翻/高拉的价值在"爆发力 + 后链激活"，不在"最大重量"
2. **重量不宜过大**：建议 ≤ 硬拉 1RM 的 60%，避免疲劳叠加
3. **与硬拉主项的间隔**：膝上动作结束后，休息 5-10 分钟再开始硬拉主项
4. **MRV 审计**：膝上高翻/高拉计入"后链"容量，需与 RDL、Good Morning 等动作叠加审计
5. **不是必须动作**：若训练者硬拉启动无问题且恢复良好，可以不用膝上变式，直接用传统热身 + 硬拉主项

## 奥举辅助的 MRV 审计

**注意**：奥举动作也会占用 MRV 配额！

**审计示例**：
```
用户：男性，30岁，训练3年
高翻安排：3×5 @ RPE 7（容量期）

后链容量审计：
- 深蹲主项：5组（股四，非后链）
- RDL：2组（后链）
- 高翻：3组（后链 + 全身）
- 总计后链：5组 < MRV 16组 ✅ 通过
```

**调整原则**：
- 若后链容量接近 MRV（≥80%）→ 减少 RDL 组数，保留高翻
- 高翻的「技术价值」> RDL 的「容量价值」（对全身爆发力）

## 常见错误

### 薄弱点分析

1. ❌ 所有薄弱点都练 → ✅ 优先解决 1-2 个最关键薄弱点
2. ❌ 辅助动作重量太重（RPE 9+） → ✅ 辅助动作 RPE 7-8，技术优先
3. ❌ 忽略技术细节（如暂停节奏） → ✅ 录像复盘，确保技术正确

### 奥林匹克举重

1. ❌ 高翻用 90% 1RM 堆容量 → ✅ 高翻 RPE ≤ 8，技术 > 重量
2. ❌ 硬拉日做高翻（下背疲劳叠加） → ✅ 高翻安排在深蹲日或独立日
3. ❌ 抓举技术不熟练就用大重量 → ✅ 先练高抓（Power Snatch）或 Clean Pull
4. ❌ 忽略奥举的 MRV 占用 → ✅ 审计时计入奥举组数
