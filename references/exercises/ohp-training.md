## 实力推（Overhead Press, OHP）

### 为什么在力量训练中加入 OHP？

**价值**：
1. **肩部整体发展**：前束/中束/后束 + 肩袖肌群
2. **核心稳定性**：推举时核心必须刚性支撑
3. **全身协调性**：与深蹲/硬拉形成「全身力量三角」
4. **功能性**：日常生活中推物过头顶的能力

**适用人群**：
- 希望全面发展上半身力量
- 非纯力量举竞技（力量举只有卧推，无 OHP）
- 希望改善肩部稳定性和健康

---

### OHP 变式与辅助动作

| 动作名称 | 中文 | 技术要点 | 适用场景 |
|---------|------|---------|---------|
| **Overhead Press (OHP)** | 实力推（站姿推举） | 站姿，核心刚性，臀/腿驱动辅助 | 主项或辅助 |
| **Push Press** | 借力推 | 腿驱动 + 髋伸展辅助，能推更大重量 | 力量期/冲刺期辅助 |
| **Push Jerk** | 炸弹推（借力推举） | 推至头顶后，髋/膝快速弯曲吸能（接铃） | 奥举风格，技术难度高 |
| **Superman Press** | 超人推 | 俯身姿势，强调肩袖稳定性 | 康复/预防性训练 |
| **Seated Dumbbell Press** | 坐姿哑铃推举 | 消除腿驱动，纯肩部力量 | 容量期辅助 |
| **Pavlo Press** | 帕夫罗推 | 空杆或轻重量，肩袖预热 | 每次训练前热身 |
| **Face Pull** | 面拉 | 绳索拉向面部，强化肩袖和后束 | 每次训练后辅助 |

### OHP 在周期中的安排

#### 作为主项（替代卧推或加入训练日）

**适用用户**：非力量举竞技，希望全面发展上半身

**安排示例**（4天/周）：
```
周一：深蹲 + OHP + 上身拉
周三：卧推 + 暂停卧推 + 夹胸
周五：硬拉 + OHP（轻） + 划船
周日：上身推辅助 + 核心 + 有氧
```

#### 作为辅助动作（大多数力量举运动员）

**容量期 W1-3**：
- OHP：3×8 @ RPE 7（作为肩部辅助）
- 或 Push Press：3×5 @ RPE 7-8（爆发力）

**力量期 W5-6**：
- OHP：2×5 @ RPE 8
- 或 Push Press：2×3 @ RPE 8.5

**冲刺期 W7-8**：
- OHP：1×3-5 @ RPE 6-7（降容保恢复）
- 或移除 OHP，优先主项恢复

### OHP 技术要点

#### 标准 OHP（站姿实力推）

**动作阶段**：
1. **起始**：杠铃置于锁骨/前肩，握距略宽于肩，核心收紧，臀部夹紧
2. **推起**：腿/臀微驱动（不是借力推），杠铃贴脸直线推起
3. **锁定**：肘部完全伸展，头部微前伸（让杠铃通过），核心保持刚性
4. **下放**：控制下放，杠铃贴脸回到锁骨

**关键提示**：
- **核心刚性**：推举时臀部不能松动，腰背不能过度伸展（避免腰椎压力）
- **腿驱动 vs. 借力推**：OHP 只允许「微驱动」，Push Press 才是明显腿驱动
- **杠铃轨迹**：贴脸直线向上，不是弧线

#### Push Press（借力推）

**与 OHP 的区别**：
- **明显腿驱动**：髋/膝快速伸展，将杠铃「弹」离肩膀
- **重量更大**：通常比 OHP 多 10-20% 重量
- **技术性**：需要协调腿/髋/肩/臂的爆发力

**适用场景**：
- 力量期/冲刺期辅助（爆发力）
- **不能替代 OHP**：Push Press 更多是「全身爆发力」，OHP 是「纯肩部力量」

#### Push Jerk（炸弹推/借力推举）

**与 Push Press 的区别**：
- Push Press：推至锁定位置（手臂完全伸展）
- Push Jerk：推至头顶后，**髋/膝快速弯曲**（接铃），类似高翻的接铃动作

**技术难度**：高（需要奥举技术基础）

**建议**：除非有奥举经验，否则优先选 Push Press

### OHP 的 MRV 审计

**肩部 MRV 基准**：≈ 24 组/周

**审计示例**：
```
用户：男性，30岁，训练3年
OHP 安排：3×8 @ RPE 7（容量期）

肩部容量审计：
- OHP：3组
- 帕夫罗推：3组（热身）
- 面拉：3组（辅助）
- 总计：9组 < 24组（MRV）✅ 通过
```

**调整原则**：
- 若肩部容量接近 MRV（≥80%）→ 减少面拉或帕夫罗推组数
- OHP 作为主项时，占用更多 MRV 配额

## 常见错误

### OHP
1. ❌ OHP 时腰背过度伸展 → ✅ 核心刚性，臀部夹紧
2. ❌ Push Press 代替 OHP → ✅ 两者互补，不能互相替代
3. ❌ OHP 重量过大（RPE 9+） → ✅ OHP 作为辅助时 RPE 7-8
